# FS Media Engine

## Purpose

FS Media Engine은
음악, 이미지, 영상 작업을 위한
`canon-safe brief and review engine`이다.

이 엔진은 현재 단계에서
대량 생성보다 `검수와 브리프`를 우선한다.

## Core Principle

1. Lore Engine 위에서만 작동한다.
2. Story Engine보다 앞서지 않는다.
3. Reveal Control 없이 어떤 미디어 브리프도 확정하지 않는다.
4. 설정이 큰 만큼 `직접 제작`보다 `체크 가능한 브리프`를 우선한다.

## Engine Structure

### Shared Core

1. `Canonical Brief Core`
2. `Reveal Control Layer`

### Music

1. Emotion Arc Check
2. Culture / Instrument Check
3. Density / Restraint Check

### Image

1. Visual Canon Check
2. Character-to-World Binding Check
3. Style Conflict Check
4. Motif / Repetition Check

### Video

1. Scene Objective Check
2. Shot Grammar Check
3. Spoiler / Reveal Check
4. Continuity Check

## Priority

현재 단계의 우선순위:

1. 브리프 구조 만들기
2. 금지 요소 / 스포일러 통제
3. 시각 / 음악 톤 충돌 방지
4. 나중에 다른 제작 환경으로 넘길 수 있는 정리된 입력 만들기
