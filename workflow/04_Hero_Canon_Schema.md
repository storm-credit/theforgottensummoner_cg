# Hero Canon Schema

`14. 인물 백과`는 전기문 모음이 아니라 관계 중심 기록 보관소로 본다.

## Hard Canon Fields

- 이름
- 현재 분류
  - 성장 영웅
  - 현존 영웅
  - 소환 영웅
- 현재 소속 또는 무소속 상태
- 관계 핵심 태그
- 액트상 역할
- 인간적 결핍 또는 선택의 비용

## Soft Canon Fields

- 상세 전투 스타일
- 세부 장비
- 세부 연표
- 정확한 연령
- 세부 출신지
- 세부 위계

## Deferred Fields

- 완전한 전기
- 모든 전투 경력
- 완전한 수치화
- 모든 과거 인연의 확정 서술

## Required Relationship Block

모든 핵심 인물 문서는 아래 질문에 답할 수 있어야 한다.

- 누구와 같은 편인가
- 누구와 한때 같은 편이었는가
- 누구와 적대하는가
- 누구를 소문으로만 아는가
- 누구와 스쳐 지나갔는가
- 누구와 과거에만 엮였는가
- 누구와 미래에 크게 얽힐 가능성이 있는가

## Anti-Patterns

- 강함만 있고 인간적 비용이 없음
- 모든 중심 사건과 연결됨
- 관계가 전부 즉시 명확함
- 세력 문서와의 앵커가 없음
