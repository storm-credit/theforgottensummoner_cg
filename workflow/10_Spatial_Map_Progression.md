# Spatial Map Progression

## Goal

세계지도, 도시 지도, 건물 내부 맵은 한 번에 완성하지 않는다.

대신 설정집 정리와 같이 가면서 아래 순서로 점진적으로 만든다.

1. 세계 축
2. 대륙 축
3. 도시 축
4. 건물 축
5. 실내 동선 축

## Why Gradual

- 세계관과 세력이 안정되기 전에 지도를 먼저 고정하면 나중에 다시 뜯어야 한다.
- 반대로 지도 축이 전혀 없으면 세력과 인물 이동 동선이 붕 뜬다.
- 그래서 지도는 `메인 진행을 보조하는 점진적 구조`가 맞다.

## Map Layers

### 1. World Map

- 대륙 배치
- 항로
- 국경성 갈등축
- 장거리 이동 병목
- 범대륙 확장과 분리된 핵심 무대

### 2. Continental Map

- 대륙별 권역
- 핵심 국가/세력 위치
- 변경 지대
- 길드/상단/부족 활동권
- 이동 난이도

### 3. City Map

- 수도
- 자유도시
- 항구
- 시장
- 빈민가
- 귀족가
- 종교 구역
- 공방 / 길드 지부

### 4. Landmark / Building Map

- 왕성
- 마탑
- 신전
- 요새
- 길드 본부
- 금고
- 연구소

### 5. Interior Route Map

- 입구
- 경비 루트
- 비밀 통로
- 금고 / 제단 / 서고 / 실험실 위치
- 전투 / 잠입 / 추격에 쓰이는 동선

## Recommended Timing

### Now

- 세계지도: 축만 메모
- 대륙 지도: 핵심 권역만 메모
- 도시 / 건물 / 실내: 아직 확정하지 않고 backlog로 관리

### After Continent Frame

- 각 대륙의 대표 도시와 권역 배치
- 부족 활동권, 가문 권역, 길드 거점 표시

### After Faction Stabilization

- 핵심 세력 본거지 지도
- 교역 루트
- 전쟁 루트
- 밀수 / 정보망 루트

### After Character and Relationship Layer

- 주요 사건 이동 동선
- 스쳐 지나감 / 엇갈림 / 재회가 가능한 동선 설계

## Minimum Questions Per Map

1. 누가 여기서 사는가
2. 누가 여기로 오려 하는가
3. 누가 여기서 쫓겨나는가
4. 왜 이 지형이 서사에 필요한가
5. 이동 경로가 인간 관계와 갈등을 만들 수 있는가

## Do Not Do

- 예쁜 지도부터 만들기
- 대륙 구조가 안 잡혔는데 도시 지도를 먼저 확정하기
- 건물 내부를 게임 던전처럼만 설계하고 생활 동선을 잊기
- 세력과 무관한 랜드마크를 남발하기

## Current Rule

- 지도 축은 메인 진행의 보조선이다.
- 메인 순서는 `세계관 -> 대륙 -> 가문/부족/길드 -> 세력 -> 인물/관계`를 유지한다.
- 지도는 각 단계가 끝날 때 필요한 만큼만 같이 채운다.
