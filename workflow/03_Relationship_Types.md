# Relationship Types

이 작품의 관계는 단순한 `아군/적군`이 아니라 시점과 거리감이 다른 연대기적 인연으로 본다.

## Core Types

- `ally`: 현재 같은 편
- `former_ally`: 과거에는 같은 편이었으나 지금은 아님
- `rival`: 적대까진 아니지만 긴장과 경쟁이 지속되는 관계
- `enemy`: 현재 명확한 적대 관계
- `rumor_only`: 서로 직접 만나지 않았고 소문이나 기록으로만 존재를 앎
- `crossed_paths`: 직접 스쳤거나 같은 사건을 다른 각도에서 지나감
- `past_only`: 과거 시점에서만 강한 접점이 있고 현재는 멀어짐
- `future_only`: 현재는 약하지만 향후 결정적 접점이 예정된 관계
- `asymmetric_awareness`: 한쪽만 상대를 알고 다른 쪽은 모름
- `same_side_different_front`: 같은 편이지만 다른 전선, 다른 이해관계로 멀리 떨어진 관계
- `intermittent_alignment`: 같은 편이었다가 다른 편이 되었다가 다시 겹칠 수 있는 가변적 관계

## Recording Rules

- 관계는 가능한 한 단일 상태가 아니라 `현재 상태 + 과거 흔적 + 미래 가능성`으로 기록한다.
- `rumor_only`와 `crossed_paths`는 약한 관계가 아니라 중요한 서사 자산으로 취급한다.
- 관계가 확정되지 않으면 삭제하지 말고 `Soft Canon`으로 둔다.
- 관계가 강할수록 만남의 횟수보다 감정 비용과 선택의 흔적을 기록한다.

## Audit Questions

- 이 관계는 우연한 접촉인가, 액트 전체에 흔적을 남기는가
- 관계 변화에 비용이 있는가
- 같은 편과 다른 편 사이 이동이 작위적이지 않은가
- 직접 만남 없이도 서사적으로 유효한가
