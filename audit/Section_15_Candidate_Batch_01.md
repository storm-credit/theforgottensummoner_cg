# Section 15 Candidate Batch 01

이 문서는 `15번 인물 백과`로 회수할 가능성이 높은
첫 조직망 인간층 배치를 정리한 시트다.

## Batch Theme

- `그림자 첩보망`
- `철의 금융 연맹`
- `침묵의 상회`

즉 국가/가문보다
정보, 금융, 암시장, 비밀 유통 쪽 인간층을 먼저 회수하는 배치다.

## Candidate Set

### 1. 그림자 첩보망 계열

- `이리나 폰 루즈`
  - current status:
    - `14 보류`
  - reason:
    - 이미 고아 인물로 독립 문서가 있고, 정보망의 중심 인물로 반복 연결된다.

- `거울방 무한 성채 운영자 계열`
  - bucket:
    - `Organization Mesh`
  - reason:
    - 특정 영토 없는 이공간 거점 운영 인물은 세계의 숨은 인간층으로 유용하다.

- `0차원 벌레/쥐 떼 통신망 조율자 계열`
  - bucket:
    - `Organization Mesh`
  - reason:
    - 정보망의 실제 현장 인간 얼굴을 붙일 수 있다.

- `환상 암살단 현장 집행자 계열`
  - bucket:
    - `Organization Mesh`
  - reason:
    - 암살/환청/정신 타격조를 담당하는 중간 간부층으로 회수 가치가 높다.

### 2. 철의 금융 연맹 계열

- `칼리크 디트리히`
  - current status:
    - `14 보류`
  - reason:
    - 금융-암시장-유통 연결의 핵심 얼굴이라 회수 우선도가 높다.

- `채무 집행관 계열`
  - bucket:
    - `Organization Mesh`
  - reason:
    - 전쟁과 국가를 빚으로 지배하는 구조를 인간 얼굴로 보여줄 수 있다.

- `기축 통화 보관인 / 채권 관리자 계열`
  - bucket:
    - `Organization Mesh`
  - reason:
    - 금융 조직을 그냥 개념이 아니라 실제 인간 층으로 보이게 해준다.

### 3. 침묵의 상회 계열

- `네크로 우물 현장 연락책 계열`
  - bucket:
    - `Organization Mesh`
  - reason:
    - 묘지와 카타콤 기반 연락망을 현장 인물로 구현할 수 있다.

- `절대 보안 암살 청부 접수자 계열`
  - bucket:
    - `Organization Mesh`
  - reason:
    - 의뢰-청부-암살 구조의 관문 역할을 한다.

- `사령 기반 텔레파시망 관리자 계열`
  - bucket:
    - `Organization Mesh`
  - reason:
    - 조직의 기이한 통신 구조를 인물화하는 데 유리하다.

## Priority

1. `이리나 폰 루즈`

## Conductor Note

첫 배치에서 군주나 장군보다
이런 인간층을 먼저 회수하는 이유는 간단하다.

- `14번` 영웅의 배경을 떠받치는 실무 인간층이 필요하고
- 세계가 실제로 돌아가는 감각은 이런 사람들에서 나오기 때문이다.
