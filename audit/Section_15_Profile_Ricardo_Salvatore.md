# 리카르도 살바토레

## 1. Core Identity

- primary anchor:
  - `숨은 거래단`
- role:
  - `거래의 감시자`
- type:
  - `Organization Mesh`
- status:
  - `safe_15`

## 2. World Function

- 자유도시의 그림자 거래가
  단순 합의가 아니라 `위반 시 반드시 대가가 돌아오는 체계`임을 보장하는 인물이다.
- 장부에 적힌 계약을 실제 공포로 번역하는
  중재 타격대의 중심 축이다.
- 영웅보다 실무 권력자에 가깝지만,
  이 인물이 있어야 숨은 거래단의 질서가 허상으로 보이지 않는다.

## 3. Spatial Tie

- primary place:
  - `거래 중재장`
- secondary place:
  - `보복 집행 구역`
- route value:
  - `중재장 / 심문실 / 응징 통로 / 거래 위반 인계선`

## 4. Relation Hooks

- `벤투라 모레티`
  - `ally`
  - 거래를 여는 자와 거래를 지키게 하는 자의 짝이다.
- `엘리자베스 로슈포르`
  - `ally`
  - 장부에 기록된 위반 사항을 집행 근거로 삼는다.
- `에블린 더모트`
  - `intermittent_alignment`
  - 위장 신분으로 도망친 거래 파기자를 추적하는 축이 된다.
- `세실리아 메르카토르`
  - `enemy`
  - 법적 중재와 그림자 보복 질서가 충돌하는 상대다.
- `장물 거래자 / 도피자`
  - `enemy`
  - 숨은 거래단의 규칙을 어긴 자들에게 공포의 얼굴이 된다.

## 5. Archive Use

- 숨은 거래단 묶음에서
  `거래 위반의 대가`를 설명하는 핵심 집행 인물이다.
- 조직이 단순 정보망이 아니라
  보복 체계를 가진 도시 권력이라는 걸 보여준다.
- 심문실, 중재장, 그림자 법정, 응징 동선 같은 공간 서사에 강하다.
