# Continuous Workstream

이 문서는 `오늘 할 일`이 아니라
현재 오케스트라가 전체 작업을 어떤 순서로 밀고 있는지 보여주는
연속 진행 기준서다.

중간에 끊지 않고
`구조 -> 판정 -> 회수 -> 보강 -> 지도` 순서로 이어서 간다.

1. `14번` 구조 충돌 판정

5. `15번`을 `Named Notables`와 `Operational Lines`로 분리

## Sequential Order

### Stage 1. Core Structure

- 세계관 코어
- 대륙 프레임
- 가문 / 부족 / 길드 충분성
- 세력 spine

### Stage 2. Hero Archive Logic

- `14번` 루트 정리
- `8번 ↔ 14번` 링크맵
- 관계 타입 표준화
- macro overlink 감시

### Stage 3. Mid-Human Recovery

- `15번` intake 구조 확정
- 후보 등록부 작성
- 중복 신호 검증
- 우선순위 정렬
- Tier별 intake 카드 제작

### Stage 4. Spatial Support

- 자유도시 도시 시드
- 운영 매트릭스
- 낮/밤 이중 도시망
- 항만 / 용병 / 암시장 / 비밀 거래장 동선

### Stage 5. Later Expansion

- 범대륙 후기 확장 재설계
- 아이템 축 본격 확장
- 액트별 관계망
- 개연성 보강
