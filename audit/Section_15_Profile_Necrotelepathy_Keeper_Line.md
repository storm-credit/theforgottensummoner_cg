# 사령 기반 텔레파시망 관리자 계열

  - `침묵의 상회`
- role:
  - `사령 통신 유지 관리자`
- type:
  - `Organization Mesh`
- status:
  - `safe_15`

## 2. World Function

- 침묵의 상회가
  단순 암살 청부 조직이 아니라 죽은 자의 잔향까지 통신망으로 활용하는
  비정상적 운영 체계를 가졌다는 걸 보여주는 실무 계열이다.
- 말하는 자보다 `울리는 자`에 가깝고,
  살아 있는 전달보다 사령 공명을 더 신뢰하는 연락 질서를 유지한다.
- 세계관의 괴이한 면을 과장하지 않으면서도
  음지 문명의 깊이를 살리는 데 좋다.

## 3. Spatial Tie

  - `사령 공명실`

## 4. Relation Hooks

- `네크로 우물 현장 연락책 계열`
  - `ally`
  - 현장 연락과 사령 공명이 맞물린다.
- `거울방 무한 성채 운영자 계열`
  - `intermittent_alignment`
  - 기억 보관과 죽은 통신이 교차하는 기괴한 접점이다.
- `칼리크 디트리히`
  - `intermittent_alignment`
  - 금융, 암살, 은폐가 얽히는 어두운 상층 연결이 가능하다.
- `침입자 / 배신자 / 의뢰 파기자`
  - `enemy`
  - 지워진 목소리로까지 추적당하는 대상들이다.

## 5. Archive Use

- 침묵의 상회를 구체적인 공간과 기술을 가진 조직으로 만든다.
- 사령 공명실, 묘지 연락 회랑, 잔향 봉인실 같은 공간 서사에 강하다.
- 괴이한 통신망, 죽은 기록, 음지 연락 문명을 보강하는 데 유용하다.

## 3-1. Policy Guard

- 이 관리자 계열이 여는 `사령 공명실 / 잔향 봉인실 / 묘지 연락 회랑` 축은
  괴이한 음지 조직의 비정상 통신 제도를 보여주는 증거다.
- 다만 이를 전통 귀족국가형 `state_house strong`이나 정상 국가기관 복원 근거로 읽지는 않는다.
- 즉 이 계열은 `dark institution / occult communications` 범위에서만 읽는다.
