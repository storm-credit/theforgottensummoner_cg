# FS Cross-Chronicle Firewall

이 문서는 다른 크로니클의 톤, 어휘, 서사 문법이
아스트라리스 연대기 판타지 버전으로 새어 들어오지 않게 막는 방화벽이다.

## Scope

현재 방화벽 대상:

- 동양풍 / 무협풍
- 다른 연대기의 고유 작법 문법
- 현대 기업물 / 범죄물 어휘 과다 유입
- 초월전 / 신격전 중심의 과열 서사

## Allowed Tone

- 서양 정통 에픽 판타지
- 인간 중심 군상 서사
- 대륙, 세력, 기사단, 길드, 공방, 서고, 성소, 항구 중심 질감
- 라틴/그리스/북유럽/산스크리트 계열 이름감

## Blocked Drift

| Drift Type | Block Rule | Example |
|---|---|---|
| `동양 / 무협 어휘` | 아스트라리스 본편 설정집에 넣지 않는다 | 문파, 강호, 협객, 내공, 기연 |
| `다른 크로니클 전용 질감` | 차용 전 `hold`에 둔다 | 다른 연대기의 종교/세계법칙/전용 위상 |
| `현대 범죄조직 직설명` | 바로 정본명으로 승격하지 않는다 | cartel, syndicate, mafia 직역 |
| `메가코프식 기업어` | display canon 완화 전 사용 금지 | megacorp, corporation 직설 사용 |
| `신격전 과열` | 인간 서사보다 앞서면 차단 | 신과 전면전, 우주 멸망 중심 전개 |

## Routing Rule

새 설정이 아래에 걸리면 바로 `hold` 또는 `tone_check_required`로 둔다.

- 다른 크로니클 고유 용어
- 무협/동양 문법
- 지나치게 현대적인 직설 명칭
- 인간 서사보다 초월 위상을 먼저 강조하는 설정

## Conductor Note

이 방화벽은 창작 억제가 목적이 아니다.

목적은
`이 크로니클이 무엇이 아닌가`를 잊지 않게 하는 것이다.
