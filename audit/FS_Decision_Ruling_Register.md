# FS Decision Ruling Register

이 문서는 Conductor가 내린 중요한 판정을 짧게 남기는 장부다.

목적:

- 같은 문제를 매번 처음부터 다시 판정하지 않는다.
- 판정 결과뿐 아니라 `왜 그렇게 봤는지`를 짧게 남긴다.

## Rule

- 길게 서술하지 않는다.
- 한 항목당 `대상 / 판정 / 이유 / 후속`만 남긴다.
- 충돌 문서가 있으면 해당 문서명을 같이 적는다.

## Current Rulings

| Target | Ruling | Why | Follow-up |
|---|---|---|---|
| `14 / 15 split` | `14 = 서사 중심 영웅`, `15 = 명사형 영웅 + 유명 NPC + 중간 인물` | 영웅백과 비대화 방지, 세계 명사층 보존 | 지속 적용 |
| `15 routing` | `대륙 -> 세력 / 도시 / 조직`을 본체로 쓴다 | 직업별 폴더링은 세계 spine을 무너뜨림 | 지속 적용 |
| `범대륙 후기 확장` | `deferred expansion` | 후기 증설 구역, 메인 5대륙 안정화가 먼저 | 후순위 유지 |
| `동양 / 무협 요소` | `아스트라리스 판타지 버전에서는 차단` | 다른 크로니클과 분리 | Firewall에서 지속 감시 |
| `울프가르 드래곤포지` | `named_notable_candidate / grade_caution` | A급/영웅록 신호 존재 | 14 독립 파일 여부 후속 확인 |
| `에리온 드라코비스` | `named_notable_candidate / grade_caution / name_collision_watch` | A급/세력 핵심표 + Erion 충돌 | 병합 금지 유지 |
| `오그마` | `named_notable_candidate` | 명사형 NPC 성격이 가장 선명 | 유지 |
| `프로스트 명사층` | `stable candidate보다 role slot 우선` | 장소-기관 기능이 인물보다 강함 | Slot Maturation으로 추적 |

## Conductor Note

이 장부는 `최종 정답` 장부가 아니다.

대신
`지금까지 어떻게 판단해 왔는가`를 잃지 않게 하는 장부다.
