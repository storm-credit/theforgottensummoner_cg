# FS Decision Ruling Register

이 문서는 Conductor가 내린 중요한 판정을 짧게 남기는 장부다.

목적:

- 같은 문제를 매번 처음부터 다시 판정하지 않는다.
- 판정 결과뿐 아니라 `왜 그렇게 봤는지`를 짧게 남긴다.

## Rule

- 길게 서술하지 않는다.
- 한 항목당 `대상 / 판정 / 이유 / 후속`만 남긴다.
- 충돌 문서가 있으면 해당 문서명을 같이 적는다.
