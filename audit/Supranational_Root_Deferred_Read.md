# Supranational Root Deferred Read

이 문서는 `범대륙 초국가 및 중립` 루트를
`display canon / deferred expansion` 관점으로 다시 읽은 판정표다.

## Conductor Verdict

범대륙 루트는 보존한다.

하지만 지금은 메인 5대륙 spine보다 후순위다.
실제 파일명이나 원본 경로를 바꾸지 않고,
`표면 표시명 후보`와 `기능 분류`만 관리한다.
