# Section 15 Named Notable Template

이 문서는 `15번 Named Notables`에 들어갈
이름 있는 명사형 인물 시트의 공용 템플릿이다.

이 축은 `Operational Lines`와 다르게
개별 이름과 기억 가치를 우선한다.

## Template

```md
# [Name]

## 1. Core Identity

- reading:
  - `대표 호칭`
- anchor:
  - `도시 / 세력 / 학파 / 공방`
- type:
  - `Named Notable`
- current judgment:
  - `strong_15 / borderline_14_15 / hold`

## 2. Why Remembered

- 왜 이 인물을 사람들이 기억하는가
- 왜 찾아가야 하는가
- 세계 안에서 어떤 분야의 얼굴인가

## 3. World Function

- 지식, 제작, 기록, 치료, 감정, 전승 등
  무엇을 여는 인물인지
- 주인공이 아니어도 왜 중요한지

## 4. Spatial Tie

- primary place:
  - `핵심 장소`
- secondary place:
  - `부차 장소`
- route value:
  - `서고 / 공방 / 제단 / 탑 / 항만 / 진료실` 등

## 5. Relation Hooks

- 어떤 영웅이나 세력과 닿는가
- 조언자, 거래자, 경고자, 연구자, 복원자 같은
  관계 역할이 무엇인가

## 6. Archive Use

- `14`보다 `15 Named Notables`에 두는 이유
- 아이템, 지도, 지식, 전승, 도시 분위기 중 어디에 강한지

## 7. Conductor Note

- 지금 확정할 것
- 14와 겹칠 위험
- 나중에 더 읽어야 할 것
```
