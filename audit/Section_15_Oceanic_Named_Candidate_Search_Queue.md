# Section 15 Oceanic Named Candidate Search Queue

이 문서는 해양 대륙의 `15 Named Notables` 후보를
실제 named candidate 탐색 단계로 내리기 위한 큐다.

## Rule

- 새 이름을 즉시 발명하지 않는다.
- 먼저 원본 참조 문서에서 실제 이름이 있는지 찾는다.
- A급, SS급, 히어로급, 제독, 단장 신호가 있으면 `14` 확인 전 15로 확정하지 않는다.
- 작업용 라벨과 표면명 후보를 분리한다.

## Search Order

1. `바다의 교단 / 오라클 바지`에서 수석 오라클 후보를 찾는다.
2. `황금 함대 / 골든 앵커 하버`에서 항로 기록관과 재무 장부층을 찾는다.
3. `크로스윈드 포트 / 왕실 조선소`에서 조선소 수석장 후보를 찾는다.
4. `블랙워터 항구 / 붉은 해골 섬`에서 감정관 후보를 찾는다.
5. `볼트 오브 아우룸 / Abyssal Vaults`에서 금고/장부 보관층 후보를 찾는다.

## Conductor Note

이 큐는 `15번` 인물 생성표가 아니라 탐색표다.

그 뒤에만 `15 Named Notables` 후보로 올린다.
