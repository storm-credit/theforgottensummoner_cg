# 땅굴쥐 운반책 계열

## 1. Core Identity

- primary anchor:
  - `포트 넥서스 / 지하 운송망`
- role:
  - `지하 운송 실무자`
- type:
  - `Region Embedded`
- status:
  - `safe_15`

## 2. World Function

- 자유도시의 그림자 경제가
  결국 누군가의 어깨와 발로 움직인다는 걸 보여주는 가장 현장적인 운반 계열이다.
- 하수구, 지하 굴, 비밀 창고, 제3 부두 사이를 오가며
  작은 물건부터 금지 문서, 약물, 인장, 피 묻은 장부까지 운반한다.
- 거물들의 이름만으로는 보이지 않는
  도시 밤세계의 물리적 노동을 복구한다.

## 3. Spatial Tie

- primary place:
  - `지하 운송선`
- secondary place:
  - `하수구 교차점`
- route value:
  - `지하 통로 / 하수구 / 부두 우회선 / 비밀 창고 연결선`

## 3-1. Policy Guard

- 이 계열은 자유도시 `shadow_port / urban_market`의 지하 운송 실무를 보강한다.
- 땅굴 물류와 하수구 운반은 도시 밤세계의 물류 기능 구획이지,
  해양 `토착 공동체층` 본체나 혈통 공동체 복원 근거가 아니다.

## 4. Relation Hooks

- `모이라 와일드웨이브`
  - `ally`
  - 항만 그림자 거래가 실제 물류로 연결된다.
- `벤투라 모레티`
  - `intermittent_alignment`
  - 익명 거래의 물적 운반을 맡을 수 있다.
- `땅굴쥐 운반책 계열`
  - `enemy`
  - 같은 하부망 안에서도 배신과 강탈이 잦을 수 있다.
- `그림자 정보단 현장 관리자 계열`
  - `intermittent_alignment`
  - 물류선과 정보선이 같은 지하 통로를 공유할 수 있다.

## 5. Archive Use

- 자유도시 지하 운송망을 `사람이 뛰는 구조`로 복구한다.
- 하수구, 교차점, 지하 창고, 제3 부두 동선을 살리는 데 유용하다.
- 추적, 잠입, 운반 사고, 물류선 배신 서사에 강하다.

## 6. Conductor Note

- 이 시점에 확정할 것:
  - 지하 운송 실무 계열이라는 점
  - 지하 운송선과 하수구 교차점에 강하게 묶인다는 점
  - 밤세계의 가장 물리적인 노동층이라는 점
- 후속 보강축:
  - 전달 암호
  - 운반 단가
  - 배달 실패와 시체 처리 규칙
- 14와 충돌 가능성:
- 작업본 기준 `매우 낮음`
