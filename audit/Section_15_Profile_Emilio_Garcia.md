# 에밀리오 가르시아

## 1. Core Identity

- primary anchor:
  - `철혈 보증단`
- role:
  - `강제 계약자`
- type:
  - `Organization Mesh`
- status:
  - `safe_15`

## 2. World Function

- 자유도시의 검은 금융이
  단순 빚 회수에 그치지 않고 `영혼까지 묶는 계약`으로 확장된다는 걸 보여주는 인물이다.
- 로자가 거리 공포를, 도미닉이 체제 폭력을 맡는다면
  에밀리오는 그 폭력의 최종 서명을 담당한다.
- 채무자와 포로, 하청 용병, 실험 재료가
  왜 빠져나오지 못하는지를 계약과 처형의 논리로 고정한다.

## 3. Spatial Tie

- primary place:
  - `강제 계약실`
- secondary place:
  - `처형실`
- route value:
  - `계약실 / 처형실 / 봉인실 / 담보 인계선`

## 4. Relation Hooks

- `도미닉 발사노`
  - `ally`
  - 회수 체계의 마지막 보증 수단으로 연결된다.
- `로자 바르톨로`
  - `ally`
  - 징수 이후에도 남는 저항을 계약과 처형으로 마무리한다.
- `로렌조 카르미네`
  - `intermittent_alignment`
  - 담보 인력이나 처분 대상이 그림자 거래로 흘러갈 수 있다.
- `채무자 / 하청 용병 / 몰락 귀족`
  - `enemy`
  - 자유를 계약서와 공포로 봉인하는 상대다.

## 5. Archive Use

- 철혈 보증단 묶음에서
  `끝맺는 자`의 역할을 맡는다.
- 금융 폭력, 처형 의식, 강제 봉인 계약 같은
  더 어두운 서사를 보강할 때 유용하다.
- 계약실, 처형실, 봉인실 같은 내부 공간 서사에 강하다.
