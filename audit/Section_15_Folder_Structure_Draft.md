# Section 15 Folder Structure Draft

이 문서는 `15 Named Notables`의 가상 폴더 구조 초안이다.

중요:

- 실제 폴더 생성/이동은 하지 않는다.
- 이 문서는 `cg` 안의 설계도다.
- 원본 저장소는 수정하지 않는다.
- 구조 기준은 직업이 아니라 `대륙 -> 세력 / 도시 / 조직`이다.

## Proposed Top Structure

```text
15. 인물백과
  15-A. Named Notables
    1. 크림슨
    2. 에테르
    3. 프로스트
    4. 해양
    5. 오벨리스크
    6. 범대륙 후기 확장
  15-B. Operational Lines
    1. 도시 운영층
    2. 항만 / 경매 / 금융 / 첩보
    3. 역할 계열 색인
  15-C. Need Named Candidate Slots
    1. 에테르 슬롯
    2. 프로스트 슬롯
    3. 해양 슬롯
    4. 오벨리스크 슬롯
```

이 초안에 대해 `Revision Gate`를 통과할 수 있는지 점검하는 것이다.
