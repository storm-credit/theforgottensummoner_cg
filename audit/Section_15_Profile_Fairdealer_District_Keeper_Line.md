# 페어딜러 상단 구역장 계열

## 1. Core Identity

- primary anchor:
  - `자유도시연합 / 페어딜러 상단`
- role:
  - `도시별 상권 관리자`
- type:
  - `Organization Mesh`
- status:
  - `safe_15`

## 2. World Function

- 자유도시의 합법 상권과 회색 상권이
  어디서 겹치고 어디서 갈라지는지를 현장 단위로 보여주는 실무 계열이다.
- 페어딜러 상단의 이상이
  실제 거리에서는 어떻게 타협과 편의, 묵인과 조정으로 바뀌는지 설명한다.
- 영웅이 아니라도
  도시마다 다른 상업 온도와 회색 거래 허용선을 잡는 데 중요하다.

## 3. Spatial Tie

- primary place:
  - `상권 회의실`
- secondary place:
  - `시장 감시 발코니`
- route value:
  - `회의실 / 시장 거리 / 세금 징수선 / 회색 거래 중재 구역`

## 4. Relation Hooks

- `레이나 브라이트헤븐`
  - `ally`
  - 공식 상권과 비공식 조정 질서가 연결된다.
- `세실리아 메르카토르`
  - `intermittent_alignment`
  - 공적 외교와 현장 상권 운영이 필요에 따라 만나고 갈라진다.
- `대륙 무역 연맹 중개자 계열`
  - `ally`
  - 범대륙 유통 질서가 도시 상권으로 내려오는 접점이다.
- `검은 경매소 경매관 계열`
  - `enemy`
  - 합법 상권과 그림자 경매가 경계선을 두고 भिड는다.

## 5. Archive Use

- 페어딜러 상단을 `영웅 몇 명의 이름`이 아니라
  도시 운영층이 있는 조직으로 보이게 한다.
- 시장 거리, 상권 회의실, 감시 발코니 같은 공간에 잘 맞는다.
- 상권 다툼, 회색 거래 묵인, 도시별 상업 분위기 차이를 보강하는 데 강하다.
