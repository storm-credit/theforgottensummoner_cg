# Section 15 Folder Draft Routing Plan

이 문서는 `15 Named Notables`의 실제 폴더링을 하기 전,
후보별 가상 라우팅을 설계하는 초안이다.

중요:

- 실제 파일 이동 계획이 아니다.
- 원본 저장소를 수정하지 않는다.
- `cg` 안에서만 라우팅 기준을 점검한다.
- 폴더 기준은 직업이 아니라 `대륙 -> 세력 / 도시 / 조직`이다.

## Revision Gate

실제 폴더 초안을 만들기 전 반드시 확인한다.

1. 14번 독립 파일 존재 여부.
2. 이름 충돌 레지스터 등재 여부.
3. 대륙 spine과 맞는지.
4. 세력/도시/조직 앵커가 명확한지.
5. `Hard Canon`이 아니라면 `draft_route` 또는 `verify_before_15`로 유지.

## Conductor Decision

아직 실제 이동은 하지 않는다.

다음 실작업은 stable 15 triad consistency lock을 바탕으로
commit/push 여부를 판단하거나 보조 후보 consistency sweep으로 넘기는 것이다.
