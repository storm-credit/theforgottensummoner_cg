# Section 15 Foldering Test - Crimson Named Notables

이 문서는 `15번 Named Notables`를 실제 폴더링할 때
기존 구조처럼 `대륙 -> 세력 / 도시 / 조직` 앵커를 유지할 수 있는지 시험하는 설계안이다.

주의:

- 원본 저장소는 수정하지 않는다.
- 아직 실제 이동/이름 변경을 하지 않는다.
- 이 문서는 `cg` 안의 라우팅 시험표다.
- 직업별 폴더링은 본체가 아니다.
- 직업과 명사 유형은 보조 색인으로만 둔다.

## Why Crimson First

크림슨은 현재 `15 Named Notables` 후보가 가장 안정적으로 보이는 대륙이다.

다른 대륙은 후보가 대부분 `verify_before_15`에 걸려 있지만,
크림슨은 아래 후보들이 비교적 15번 명사형 인물에 가깝다.

## Foldering Principle

15번 Named Notables의 본체 라우팅은 아래 순서를 따른다.

1. `대륙`
2. `세력 / 도시 / 조직`
3. `인물명`
4. 보조 태그로 `현자`, `장인`, `기록자`, `연금술사`, `공방주`, `예언자` 등을 붙인다.

즉 아래처럼 직업별 폴더를 먼저 만들지 않는다.

- `현자/에리온`
- `대장장이/울프가르`
- `고룡/오그마`

대신 아래처럼 세계 구조를 유지한다.

- `크림슨 -> 용의 후예 -> 드래곤포지 계열 -> 울프가르`
- `크림슨 -> 엘드라칸 -> 대현자층 -> 에리온`
- `크림슨 -> 엘드라칸 -> 전승 보관층 -> 오그마`
- `크림슨 -> 붉은 사막 부족 -> 현자 회의 -> 벨라나`
- `크림슨 -> 붉은 사막 부족 -> 현자 회의 -> 아리안`

## Display Name Notes

지금 단계에서는 표면 명칭을 과하게 새로 만들지 않는다.

다만 나중에 실제 문서명을 만들 때는 아래 톤을 우선한다.

| Functional Read | Better Fantasy Surface |
|---|---|
| 대장장이 | `공방주`, `용화로 장인`, `드래곤포지 장인` |
| 관리자 | `문헌지기`, `고대어 해석자`, `서고의 현자` |
| 살아있는 도서관 | `전승의 고룡`, `기억의 용` |
| 현자 회의 | `붉은 모래 현자회`, `사막의 원로회` |

금지:

- 무협식 문파/강호/내공 어휘로 바꾸지 않는다.
- 현대 조직명처럼 `카르텔`, `신디케이트`, `메가코프`를 표면명으로 쓰지 않는다.

## Conductor Decision

크림슨은 `15 Named Notables` 폴더링 시험에 적합하다.

다만 이 단계는 실제 이동이 아니라 라우팅 검증이다.

다음 액션:

1. `울프가르`, `에리온`, `오그마`를 기준으로 개별 15 시트 포맷을 시험한다.
2. `벨라나`, `아리안`은 SS/S급 핵심 인물 신호가 확인되어 시트화를 보류하고 14 경계 검증으로 되돌린다.
3. `드락사르`, `카사르`는 14 경계 큐에 유지한다.
