# 채무 집행관 계열

## 1. Core Identity

- primary anchor:
  - `철의 금융 연맹`
- role:
  - `담보 압류 담당`
- type:
  - `Organization Mesh`
- status:
  - `safe_15`

## 2. World Function

- 상층 금융 질서가 거리로 내려왔을 때
  실제로 무엇을 빼앗고 무엇을 봉인하는지 보여주는 현장 집행 계열이다.
- 집행관 계열이 큰 강제라면,
  채무 집행관은 문, 창고, 선적물, 상점, 가문 문장을 하나씩 닫아버리는 실무자다.
- 도시 경제가 왜 숨막히는지,
  상인의 일상과 귀족의 체면이 어떻게 붕괴되는지 설명해준다.

## 3. Spatial Tie

- primary place:
  - `압류 창고`
- secondary place:
  - `유통 검문소`
- route value:
  - `창고 / 검문소 / 봉인 구역 / 담보 이동선`

## 4. Relation Hooks

- `철의 금융 연맹 집행관 계열`
  - `ally`
  - 큰 강제를 실제 압류 절차로 완성한다.
- `기축 통화 보관인 / 채권 관리자 계열`
  - `ally`
  - 금고의 논리가 현장의 봉인으로 이어진다.
- `세실리아 메르카토르`
  - `enemy`
  - 상업 자유와 금융 압류 질서가 정면으로 충돌한다.
- `항구 감독관 / 지부장 / 도시 조정관 계열`
  - `intermittent_alignment`
  - 유통 차단과 항만 봉쇄 문제로 마주친다.

## 5. Archive Use

- 철의 금융 연맹 묶음에서
  가장 눈에 보이는 `빼앗는 손`을 담당한다.
- 압류 창고, 봉인 구역, 유통 검문소 같은 공간을 살아나게 한다.
- 상권 붕괴, 담보 압류, 유통 봉쇄 서사에 특히 강하다.

## 3-1. Policy Guard

- 이 계열이 여는 `압류 창고 / 유통 검문소 / 가문 문장 봉인` 축은
  오벨리스크형 제도 상층의 금융 강제력과 통제 장치를 보여주는 증거다.
- 다만 이를 전통 귀족국가형 `state_house strong` 근거로 읽지는 않는다.
- 즉 이 계열은 `nontraditional elite thin-support / coercive finance` 범위에서만 읽는다.
