# Section 8 to 15 Notable Anchor Bridge

이 문서는 `8. 세력 아카이브`의 대륙별 spine과
`15 Named Notables` 후보/역할 슬롯을 연결하는 브리지다.

목적:

- 15번 인물을 직업별로 흩뜨리지 않는다.
- 후보마다 `대륙 -> 세력 / 도시 / 조직` 앵커를 먼저 잡는다.
- 14번 영웅인지, 15번 명사형 인물인지, 아직 이름 없는 슬롯인지 분리한다.

## Routing Rule

15번 후보는 아래 순서로 라우팅한다.

1. 대륙 spine 확인.
2. 세력/도시/조직 앵커 확인.
3. 14번 영웅 신호 확인.
4. `named_notable_candidate`, `verify_before_15`, `need_named_candidate`, `keep_14` 중 하나로 상태 라벨 부여.
5. 폴더링은 `대륙 -> 세력 / 도시 / 조직` 기준으로만 한다.

## Foldering Consequence

앞으로 15번 문서 초안을 만들 때는 아래 방식으로 가정한다.

예시:

- `15 / 크림슨 / 용의 후예 / 드래곤포지 공방 / 울프가르`
- `15 / 크림슨 / 엘드라칸 / 에리온`
- `15 / 에테르 / 정령연합 / 루미라 / 엘다라`
- `15 / 해양 / 포트 아우렐리온 / 대경매장 주인 슬롯`
- `15 / 오벨리스크 / 영원의 기록탑 / 사후 서기관 슬롯`

단, 실제 폴더 이동은 아직 하지 않는다.
현재는 `cg` 안에서 라우팅 기준만 고정한다.

## Conductor Decision

15번의 다음 실작업은 새 인물을 더 찾는 것이 아니라,
위 브리지를 기준으로 프로스트 2차 narrowing까지 포함한 5대륙 closure 결과를 `15 Named Notables Anchor Map`과 다른 압축표에 같이 맞추는 것이다.

그 뒤에야 실제 폴더 초안이나 문서 이동 계획을 세운다.
