# 환상 암살단 현장 집행자 계열

## 1. Core Identity

- primary anchor:
  - `그림자 첩보망`
- role:
  - `현장 제거 집행자`
- type:
  - `Organization Mesh`
- status:
  - `safe_15`

## 2. World Function

- 첩보망이 수집과 감시에만 머물지 않고
  필요할 때는 직접 `지운다`는 사실을 보여주는 현장 집행 계열이다.
- 명단을 읽는 자가 아니라,
  명단에서 이름을 없애는 손에 가깝다.
- 도시의 보이지 않는 공포를 보강하면서도
  조직이 어디까지 넘어가는지를 선명하게 만든다.

## 3. Spatial Tie

- primary place:
  - `암시장 접선지`
- secondary place:
  - `제거 후 은닉 통로`
- route value:
  - `접선지 / 제거 통로 / 흔적 소각실 / 도주 분기점`

## 4. Relation Hooks

- `그림자 정보단 현장 관리자 계열`
  - `ally`
  - 감시와 제거가 한 체계 안에서 이어진다.
- `절대 보안 암살 청부 접수자 계열`
  - `intermittent_alignment`
  - 청부형 제거와 첩보형 제거가 겹칠 수 있다.
- `환상 암살단 현장 집행자 계열`
  - `enemy`
  - 내부 경쟁과 실패 은폐가 있을 수 있다.
- `배신자 / 폭주 정보원 / 반체제 귀족`
  - `enemy`
  - 오래 숨어도 결국 도달하는 제거 대상이다.

## 5. Archive Use

- 그림자 첩보망의 어두운 끝점을 보강한다.
- 암시장 접선지, 제거 통로, 흔적 소각실 같은 공간 서사에 강하다.
- 추적, 제거, 내부 숙청, 실패한 암살 서사에 유용하다.

## Policy Guard

- 이 현장 집행 계열이 여는 `암시장 접선지 / 제거 통로 / 흔적 소각실` 축은
  비정통 정보기관의 제거 기능과 음지 집행력을 보여주는 증거다.
- 다만 이를 전통 귀족국가형 `state_house strong` 근거로 읽지는 않는다.
- 즉 이 계열은 `nontraditional elite thin-support / covert enforcement` 범위에서만 읽는다.

- 이 시점에 확정할 것:
  - 현장 제거 집행 계열이라는 점

  - 첩보망의 제거 기능을 담당한다는 점
