# 아이언가르드 (Ironguard)

> *"강철은 녹슬지 않고, 철의 맹세는 깨지지 않는다."*

---

## 1. 기본 정보

| 항목 | 내용 |
|------|------|
| **위치** | 에테르 대륙 동부 산악 지대, 크림슨 대륙 접경 |
| **역할** | 왕국연합 최전방 군사 왕국 |
| **특징** | 철광산, 대장장이 전통, 정예 중기병 |
| **이미지** | 산 위의 석조 요새 도시, 끊임없는 망치질 소리와 군가 |

---

## 2. 지형·구조

### ■ 지리적 특성
- **산악 요새 도시**: 자연 방벽을 활용한 다층 성채
- **철광 벨트**: 도시 하부에 거대한 광산 네트워크
- **협곡 관문**: 크림슨 대륙으로 통하는 유일한 육로 통제

### ■ 주요 구역
| 구역 | 특징 |
|------|------|
| **왕성 (Iron Crown)** | 산 정상부, 철로 장식된 왕궁 |
| **대장간 거리** | 왕국 최고 무기/갑옷 생산지 |
| **병영 구역** | 상시 주둔 5만 대군 |
| **광산 지구** | 지하 채굴 도시, 민간인 다수 |
| **관문 요새** | 크림슨 방향 3중 성문 방어선 |

---

## 3. 정치·군사

### ■ 통치 구조
| 직위 | 역할 |
|------|------|
| **철왕 (Iron King)** | 군사·행정 최고 책임자 |
| **철의 원로원** | 군 장성·광산 귀족·대장장이 길드 대표 |
| **관문 총독** | 크림슨 접경 방어 전담 |

### ■ 군사력
- 왕국연합 **최강 육군** 보유
- **철의 [[01-20. 직업 & 클래스 (Classes & Jobs)/직업 도감/06. 기사 (Knight)]]단**: 중[[01-19. 아이템 도감 (Item Encyclopedia)/01-19-2. 방어구 (Armor)/12. 장갑 (Gauntlets)/12. 장갑 (Gauntlets)]] 기병 정예
- **산악 보병**: 험지 전투 특화
- **공성 병기**: 발리스타, 투석기 대량 보유

### ■ 왕국연합 내 역할
> "아이언가르드가 무너지면 연합이 무너진다"

- 크림슨 침공 방어 최전선
- 연합 군사 회의 주도권 보유
- 다른 왕국에 무기 공급
