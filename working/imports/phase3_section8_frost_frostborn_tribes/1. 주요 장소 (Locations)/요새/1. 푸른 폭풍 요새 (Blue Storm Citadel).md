# 1. 푸른 폭풍 요새 (Blue Storm Citadel)

> *"폭풍 속에서 단련되지 않은 자는 진정한 전사가 아니다. 이곳은 소년을 철인(Iron-man)으로 다시 태어나게 하는 용광로다."*
> — 훈련 교관 드라코르

---

## 기본 정보

| 항목 | 내용 |
|------|------|
| **위치** | 프로스트 대륙 동부 해안 절벽, '폭풍의 곶' |
| **역할** | 연합군 총사령부 + 정예병 훈련소 + 해안 방어 기지 |
| **별명** | "강철의 요람", "바람의 성" |
| **이미지** | 거친 파도와 눈보라가 몰아치는 절벽 위에 우뚝 솟은 검푸른 강철 요새, 수백 개의 풍차가 돌아가는 모습 |

### 핵심 시설
- 폭풍의 대장간 (Storm Forge)
- 극한 훈련소 (Limit Breaker)
- 전술 작전실 (War Room)
- 와이번 둥지 (Wyvern's Nest)
- 심해 감옥 (Deep Prison)

## 지형 및 요새 구조

### 상징적 지형

| 지형 | 설명 |
|------|------|
| **폭풍의 곶 (Storm Cape)** | 1년 내내 강력한 해풍과 눈보라가 몰아치는 악천후 지역. 서 있기만 해도 훈련이 되는 환경 |
| **풍차 방벽 (Windmill Wall)** | 요새 외벽에 설치된 수백 개의 거대한 풍차. 바람을 막는 동시에 동력을 생산하여 대장간과 기계를 돌림 |
| **공중 활주로 (Sky Runway)** | 절벽 끝에서 바다 쪽으로 길게 뻗은 비행 유닛 이착륙장. 실수하면 바로 바다로 추락 |

### 주요 구역

| 구역 | 특징 |
|------|------|
| **사령부 구역 (Command Zone)** | 중앙 탑. **[[05. 볼트가르]]**와 참모들이 작전을 수립하는 두뇌 공간 |
| **생산 구역 (Industrial Zone)** | 대장간과 공방이 밀집. 24시간 망치 소리가 끊이지 않으며, 연합군의 무기 70%를 생산 |
| **훈련 구역 (Training Zone)** | 바닷물이 들이치는 갯바위 훈련장. 맨몸으로 파도를 버티거나 절벽을 오르는 지옥 훈련 코스 |
| **항구 구역 (Port Zone)** | 절벽 아래 비밀 동굴에 숨겨진 소규모 함대 정박지 |

## 정치·행정 구조

| 기관 | 역할 |
|------|------|
| **군사 평의회** | 각 부대 지휘관들이 모여 군사 작전을 논의. 철저한 상명하복 |
| **보급청** | 막대한 양의 식량과 철광석을 관리. 횡령은 즉결 처형 |

### 권력의 현실
> "땀은 피를 아끼고, 훈련은 실수를 줄인다."

- **탈권위주의**: 이곳에서는 부족이나 가문 배경이 통하지 않음. 오직 훈련 성적과 실력만이 계급을 결정
- **독립성**: 군사 문제에 있어서는 대족장도 함부로 간섭하지 못하는 독자적인 지휘 체계 보유

| **교관단** | **[[04. 드라코르]]**가 이끄는 악마 같은 훈련 교관들. 신병들에게 공포의 대상 |

### 방어 체계
- **대공 방어**: 마법 공학으로 개량된 대형 쇠뇌(Ballista)가 하늘을 촘촘히 겨냥하고 있음
- **마법 폭풍**: 비상 시 풍차를 역회전시켜 인공적인 허리케인을 만들어 적 함대를 수장시킴

## 신앙 · 일상 · 분위기

### 신앙
- **강철과 불의 신**: 대장장이와 전사들이 믿는 투쟁적인 신앙
- **기계 숭배**: 드워프 기술의 영향으로 기계나 장비에 이름을 붙이고 애지중지하는 문화

### 일상 풍경
| 시간 | 모습 |
|------|------|
| 아침 | 기상 나팔 소리와 함께 구보하는 병사들의 우렁찬 군가 |
| 낮 | 대장간의 열기와 훈련장의 땀 냄새가 뒤섞인 치열한 현장 |
| 저녁 | 식당에서 고봉밥을 먹으며 하루의 훈련 내용을 복기하는 병사들 |
| 밤 | 등대 불빛이 바다를 비추고, 야간 불침번들의 묵묵한 순찰 |

### 요새 정서
> [!note] 분위기
> "규율과 열정."
> 삭막하고 엄격해 보이지만, 그 안에는 자신을 극한까지 단련하려는 전사들의 뜨거운 열정이 녹아 있음.

## 소환 · 균열 관련

### 공식 입장
- **무기화**: [[01-37. 소환 백과/01-13. 소환술 (Summoning System)]]을 실전적인 '전력'으로 평가함. 병사들에게 도움이 된다면 적극 도입
- **통제**: 통제되지 않는 소환수나 폭주 위험이 있는 마법은 요새 안전을 위협하므로 금지

### 실제 상황
| 계층 | 현실 |
|------|------|
| **지휘관** | 소환사를 '특수 병과'로 분류하여 전략적으로 활용하려 함 |
| **병사들** | 처음엔 낯설어하지만, 전투에서 도움을 받으면 금방 전우로 인정함 |

### [[01-15. 인물 백과 (Character Archive)/0. 주인공 일행 (Main Characters)/에반 무영 라크라시스 (Evan)]] 개입 포인트
- **[지옥 훈련 수료]**: [[01-15. 인물 백과 (Character Archive)/0. 주인공 일행 (Main Characters)/에반 무영 라크라시스 (Evan)]]이 마법을 쓰지 않고 신체 능력만으로 '극한 훈련 코스'를 완주하여 전사들의 인정을 받음
- **[[[01-34. 연금 백과/05. 연금 공정 및 영적 공명 체계/7공정]] 개선]**: 폭풍의 대장간의 비효율적인 생산 라인을 현대적 지식(분업화)으로 개선하여 무기 생산량을 2배로 늘림
- **[와이번 길들이기]**: 야생 와이번 무리가 습격했을 때, [[01-15. 인물 백과 (Character Archive)/0. 주인공 일행 (Main Characters)/에반 무영 라크라시스 (Evan)]]이 우두머리를 제압하고 길들여 '공중전 기술'을 군대에 전수
