## 4. 균형 유지 메커니즘

### 결투에 의한 서열
- 부족장에 도전하려면 **만인 앞에서 결투** — 패자는 추방 또는 사망
- "사막이 판단한다" — 자연 환경이 곧 법

### 대부족장의 무력 억지
- 잔드리크(SSS급)의 **절대 무력**이 모든 부족을 하나로 묶는 유일한 힘
- 이 억지력이 사라지면 4개 부족이 즉시 독립 → 내전

### 오아시스 통제
- 레드스톰 본부족이 **주요 오아시스를 장악** — 물이 곧 권력
- 다른 부족이 물을 얻으려면 레드스톰에 복종해야 함

### 공동의 적
- **솔라리안 제국**이라는 외부 위협이 부족들을 하나로 묶는 접착제
- 제국 위협이 사라지면 연합 유지의 명분도 사라짐

## 5. 위기 시나리오

### 시나리오 A: 잔드리크의 전사

> *"산이 무너지면, 모래알은 사방으로 흩어진다."*

- **촉발**: 잔드리크 레드스톰의 전사 또는 중상
- **전개**: 카르닥(군부) vs 벨라나(주술) vs 라카(암살) — 후계 3파전
- **위험**: 부족 연합 즉시 분열 → 솔라리안 제국이 사막 전면 정복 시도
- **외부 개입**: 용의 후예가 중재 또는 영토 침탈

### 시나리오 B: 암살 부족의 이탈

> *"그림자는 빛이 없으면 존재하지 않는다. 하지만 그림자가 빛을 삼키면?"*

- **촉발**: 라카·마라가 솔라리안 제국과 비밀 거래 → 부족 정보 유출
- **전개**: 잔드리크가 숙청 명령 → 샌드스팅어 부족 내전 or 독립 선언
- **위험**: 첩보 네트워크 상실 → 제국의 기습 공격에 무방비
- **결과**: 남은 부족이 무력으로 재통합하되, 신뢰 기반 영구 손상

### 시나리오 C: 오아시스 고갈

> *"물이 마르면, 칼이 먼저 빛난다."*

- **촉발**: 아르카나 잔재의 지맥 융탈이 사막 지하수까지 고갈
- **전개**: 물 부족 → 부족 간 오아시스 쟁탈전 → 연합 내전 발발
- **위험**: 사막 자체가 거주 불가능해지면 부족 전체 이주 → 대규모 난민
- **외부 개입**: 잊혀진 서고단이 고대 오아시스 지도를 거래 카드로 활용
