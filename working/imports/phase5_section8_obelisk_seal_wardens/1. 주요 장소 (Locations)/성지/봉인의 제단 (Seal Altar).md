# 봉인의 제단 (Seal Altar)

> *"이 제단이 빛나는 한, 봉인은 유지된다. 빛이 꺼지면... 아무도 모른다."*

---

## 1. 기본 정보

| 항목 | 내용 |
|------|------|
| **위치** | 경계의 보루 중앙, 가장 깊은 지하실 |
| **외관** | 4대 세력 문양이 새겨진 원형 제단. 중앙에 봉인 수정 |
| **역할** | 흑오벨리스크 봉인 상태 모니터링 / 봉인 강화 의식 장소 |

---

## 2. 제단의 기능

### ■ 봉인 모니터링
- 제단 중앙 수정이 **흑오벨리스크 봉인 상태**를 실시간 반영
- 밝기 감소 = 봉인 약화 경고

### ■ 강화 의식
- 4대 세력 대표가 모여 **합동 봉인 강화 의식** 거행
- 최근에는 인원 부족으로 불완전하게 진행

---

## 3. 구조

| 구역 | 설명 |
|------|------|
| **하강 계단** | 36계단. 각 계단에 과거 봉인관 이름 |
| **정화 욕조** | 의식 전 심연 오염 정화 |
| **제단 홀** | 4방향 문. 각 세력별 입장 통로 |
| **봉인 수정** | 중앙에 떠 있는 무색 수정. 봉인과 연결 |
| **기록벽** | 과거 봉인 의식 기록, 이상 징후 기록 |

---

## 4. 현재 상황

### ■ 위기 상태
- 대봉인관 실종으로 **의식 주도자 부재**
- 성국 [[01-20. 직업 & 클래스 (Classes & Jobs)/직업 도감/08. 사제 (Cleric)]] 수 감소로 정화 의식 약화
- 봉인 수정 밝기 **점점 어두워지는 중**

### ■ 비밀
- 일부 기록에 따르면, **완전 봉인 해제 시 대안** 존재 가능?
