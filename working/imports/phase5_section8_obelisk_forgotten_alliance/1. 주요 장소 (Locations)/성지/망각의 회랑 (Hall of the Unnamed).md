# 망각의 회랑 (Hall of the Unnamed)

> *"이곳에 들어오는 순간, 너는 '나'라는 이름 없이도 설 수 있는가를 먼저 시험받는다."*

---

## 1. 기본 정보

| 항목 | 내용 |
|------|------|
| **위치** | 침묵의 평원과 잊힌 시장 사이, 어느 지도에도 좌표가 찍히지 않는 안개·반향 지대 |
| **외관** | 외부에서는 돌무더기와 폐허 그림자. 의식 후 거대한 실내 공간으로 변환 |
| **역할** | 잊힌 자들의 연합 본부 / 성지 / 회의장 / 기록보관소 |

---

## 2. 구조

| 구역 | 설명 |
|------|------|
| **망각의 문** | 진입 시 이름-포기 의식. 본명을 돌판에 적어 그릇에 넣음 |
| **무명의 회의실** | 상층부 회합 공간. 본명 대신 별칭으로만 호명 |
| **실패한 역사 서고** | 공식 연대기에서 삭제된 사건만 모아둔 서고 |
| **무명자 회랑** | 얼굴 없는 초상화, 이름 지워진 위패가 늘어선 복도 |
| **잔향의 정원** | 침묵의 평원에서 흘러든 전쟁 잔향을 묶어둔 야외 정원 |
| **기억 보관실** | 한 시점의 기억/감정/결정을 봉인해 두는 곳 |

---

## 3. 입장 규칙

### ■ 망각의 문 의식
1. 자신의 본명을 입술로 속삭인다
2. 그 이름을 돌판/종이에 적어 문 옆 그릇에 넣는다
3. 회랑 안에서는 **본명을 떠올릴 수 없음** (호칭/별명/직함으로만 자신 규정)
4. 나갈 때 이름을 돌려받지만, 잊은 상태에서 한 선택/발언은 그대로 남음

| **심연 군단** | 완전한 삭제를 방해하는 존재. 겉으로는 대립 안 하지만 긴장 |
| **외부 대륙** | 지워진 진실을 사고파는 비공식 장터 역할 |
