# 그림자 은신처 (Shadow Haven)

> *"이 장소는 지도에 없다. 기억에도 없다. 하지만 필요한 자는 반드시 찾아온다."*

---

## 1. 기본 정보

| 항목 | 내용 |
|------|------|
| **위치** | 저주받은 숲 깊숙이, 유령 마을 폐허 지하 |
| **외관** | 지상에서는 무너진 건물 잔해. 지하에 거대 은신 공간 |
| **역할** | 잊힌 자들의 연합 비밀 거점 / 긴급 피난처 / 정보 교환소 |

---

## 2. 특수 기능

### ■ 존재 은폐
- 은신처 안에 있는 동안 **모든 추적 마법이 작동 안 함**
- 심연의 속삭임도 이곳까지 도달하지 못함

### ■ 이름 임시 보관
- 체류 기간 동안 자신의 이름을 **임시로 맡기고** 완전 익명 가능
- 긴급 탈출/은신 시 활용

---

## 3. 구조

| 구역 | 설명 |
|------|------|
| **폐허 입구** | 의도적으로 위험해 보이게 설계 |
| **미로 통로** | 함정과 환각으로 침입자 교란 |
| **중앙 광장** | 정보 교환, 은밀한 회의 |
| **임시 숙소** | 도주자, 부상자 임시 거주 |
| **기록 금고** | 가장 중요한 비밀 기록 보관 |
| **탈출 터널** | 3개 방향 비상 탈출로 |

---

## 4. 운영 규칙

### ■ 입장 조건
- 잊힌 자들의 연합 구성원 또는 **소개자 필요**
- 진짜 이름이 아닌 **별칭**으로만 호칭

### ■ 금지 사항
- 폭력, 추적 마법 시도, 타인 정체 캐기
- 위반 시 **기억 일부 삭제 후 추방**
