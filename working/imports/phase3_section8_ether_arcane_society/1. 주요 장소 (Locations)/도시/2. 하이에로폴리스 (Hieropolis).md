# 2. 하이에로폴리스 (Hieropolis)

> *"마법은 신비가 아니라 기술이다."*
> — 마도 공학자 길드장

---

## 기본 정보

| 항목 | 내용 |
|------|------|
| **위치** | 아르카디아 하방 지상 1km 지점 (공중 도시와 지상을 잇는 중계 도시) |
| **역할** | 마도 공학 생산 기지 + 무역 허브 + 마법 대중화의 전초기지 |
| **별명** | "톱니바퀴의 도시", "연기의 도시" |
| **이미지** | 거대한 톱니바퀴와 파이프가 얽혀 있는 스팀펑크 풍의 도시. 항상 기계 돌아가는 소리와 증기가 피어오름 |

### 핵심 시설
- 중앙 [[01-35. 마법 공학 백과/05. 마공학 기어 및 동력 체계/7기어]] 타워 (Central Gear Tower)
- 마도 열차 터미널 (Mana Train Terminal)
- 공학 연구소 (Engineering Lab)
- 인조 인간 공장 (Automaton Factory)
- 대시장 (Grand Bazaar)

## 지형 및 도시 구조

### 상징적 지형

| 지형 | 설명 |
|------|------|
| **철의 대지** | 도시 바닥이 금속판으로 포장되어 있음. 지하에는 복잡한 배관과 전선이 매설됨 |
| **스카이 리프트** | 상공의 아르카디아로 올라가는 거대한 마법 승강기. 일반인은 이용 불가 |
| **폐기물 처리장** | 도시 외곽에 쌓인 기계 부품과 마법 폐기물 산. 고물상들의 보물창고 |

### 주요 지구

| 지구 | 특징 |
|------|------|
| **공장 지구 (Factory District)** | 마법 도구(가전제품, 무기)를 대량 생산하는 공장들이 24시간 돌아감 |
| **거주 지구 (Residence District)** | 기술자와 노동자들이 사는 빽빽한 아파트 단지. 구역별로 난방과 조명이 자동 제어됨 |
| **상업 지구 (Trade District)** | 지상의 상인들이 물건을 떼어가는 도매 시장. 항상 시끄럽고 활기참 |

## 정치·행정 구조

| 기관 | 역할 |
|------|------|
| **기술자 조합 (Technocrat Union)** | 도시를 실질적으로 운영하는 기술자들의 길드. 아르카디아 7인 위원회의 하청 기관 취급을 받지만 자존심이 셈 |
| **품질 관리국** | 생산된 마도구의 성능과 안전성을 검사함 |

### 권력의 현실
> "우리는 하청업체가 아니다... 라고 말하고 싶지만."

- **예속 관계:** 아르카디아에 핵심 부품(마나 코어)을 의존하고 있어, 그들의 눈치를 볼 수밖에 없음
- **기술 유출:** 독자적인 기술력을 확보하기 위해 몰래 뒷구멍으로 외부 세력(상인 연합 등)과 손을 잡기도 함

## 군사 & 치안

| 조직 | 역할 |
|------|------|
| **오토마톤 경비대** | 입력된 매뉴얼대로만 움직이는 기계 경찰. 뇌물이 통하지 않지만 융통성도 없음 |
| **보안관** | 인간 지휘관. 기계들이 해결 못 하는 복잡한 사건을 처리함 |

### 방어 체계
- **자동 포탑:** 도시 곳곳에 숨겨진 감시 카메라와 연동된 기관총 포탑
- **전자기 펄스:** 외부 침입 시 강력한 EMP를 터뜨려 적의 마법 장비를 무력화 (아군도 마비되는 게 단점)

## 신앙 · 일상 · 분위기

### 신앙
- **기계신 (Deus Ex Machina):** 일부 과격한 기술자들은 기계가 인류를 구원할 것이라 믿는 신흥 종교를 믿음
- **효율성:** "시간은 금이다"가 곧 생활 신조

### 일상 풍경
| 시간 | 모습 |
|------|------|
| 아침 | 증기 휘슬 소리와 함께 공장 가동 시작, 출근하는 기술자들의 행렬 |
| 낮 | 마도 열차가 수시로 드나들며 물자를 싣고 내리는 분주함, 용접 불꽃이 튀는 거리 |
| 저녁 | 기름때 묻은 작업복을 입고 선술집에서 맥주를 마시는 노동자들 |
| 밤 | 공장의 야간 조명 때문에 불야성을 이룸. 골목길에서는 불법 개조 부품 거래가 이뤄짐 |

### 도시 정서
> [!note] 분위기
> "혁신과 소외."
> 최첨단 기술이 넘쳐나지만, 정작 그 기술을 만드는 사람들은 부속품처럼 취급받는 삭막함.

## 소환 · 균열 관련

### 공식 입장
- **동력원:** 소환수를 생명체라기보다 '효율적인 배터리'나 '엔진'으로 봄
- **융합:** 생체 소환수와 기계를 결합한 사이보그 몬스터 연구가 암암리에 진행됨

### 실제 상황
| 계층 | 현실 |
|------|------|
| **매드 엔지니어** | 소환수에게 기계 팔다리를 달아 전투력을 높이는 불법 투기장을 운영함 |
| **소환사** | 이곳에 오면 자신의 파트너(소환수)가 실험 재료로 쓰일까 봐 불안해함 |

### [[01-15. 인물 백과 (Character Archive)/0. 주인공 일행 (Main Characters)/에반 무영 라크라시스 (Evan)]] 개입 포인트

- **[영혼을 가진 인형]:** 폐기 처분될 위기에 처한 감정 있는 오토마톤 소녀를 [[01-15. 인물 백과 (Character Archive)/0. 주인공 일행 (Main Characters)/에반 무영 라크라시스 (Evan)]]이 구출하고, 그녀의 '인권'을 찾아줌
- **[폭주 기관차]:** 브레이크 고장 난 마도 열차를 멈추기 위해 [[01-15. 인물 백과 (Character Archive)/0. 주인공 일행 (Main Characters)/에반 무영 라크라시스 (Evan)]]이 거대 소환수를 불러 열차를 몸으로 막아세움
