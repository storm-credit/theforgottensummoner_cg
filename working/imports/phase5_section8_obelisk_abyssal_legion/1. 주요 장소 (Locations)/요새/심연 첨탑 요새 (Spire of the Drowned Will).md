# 심연 첨탑 요새 (Spire of the Drowned Will)

> *"이 탑의 꼭대기에서 하늘을 올려다보면, 끝내 땅 밑 심연이 보인다."*

---

## 1. 기본 정보

| 항목 | 내용 |
|------|------|
| **위치** | 흑오벨리스크에서 방사형 균열대 끝자락, 심연과 지상이 가장 얇게 겹친 경계 |
| **지형** | 역전된 절벽 지대. 균열 속에서 자라난 검은 기둥 |
| **역할** | 심연 군단 지상 전역 지휘 요새 / 명령·의지 전송 허브 |
| **통치자** | [[심연 군단 (Abyssal Legion)|심연 군주 (Abyssal Lord)]] 분신 + 첨탑 사령관 |

---

## 2. 요새 구조

| 구역 | 역할 |
|------|------|
| **침잠의 기단** | 지상 아래 균열 속 하부. 대규모 병력 대기실 + 의지 공유 장치 |
| **왜곡 계단** | 나선형 통로. 중력/빛/소리 감각 어긋남 |
| **심연 [[01-20. 직업 & 클래스 (Classes & Jobs)/직업 도감/03. 전령 (Messenger)]]실** | 명령 동시 전송 관제실. 벽 전체가 눈 없는 얼굴/귀만 남은 석상 |
| **의지 감금소** | 정복·동화된 영웅/왕/장군의 의지 봉인층 |
| **균열 관측실** | 상층부. 대륙 전체 균열을 지도처럼 조망 |
| **역전 정점** | 꼭대기이자 심연 최심부와 맞닿은 점. 심연 군주 상현 공간 |

---

## 3. 지휘 체계

```
심연 군주 (의지/분신/환영)
    ↓
첨탑 사령관 (Spire Warden) - 지상 전역 총지휘
    ↓
의지 번역자 (Translators of the Drowned)
```

### ■ 요새 원칙
- **"위로 갈수록 더 깊이 잠긴다"**: 상층부는 심연 의지와 동화된 자만 출입
- 지상 전쟁은 지상 기준, 심연 전쟁은 개념·의지 전쟁으로 분리 운영
