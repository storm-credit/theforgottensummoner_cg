# 지옥문 (Hell's Gate)

> *"이 문이 완전히 열리는 날, 세상은 끝난다."*

| 항목 | 내용 |
|------|------|
| **위치** | 오벨리스크 대륙 중부, 심연과 현실의 경계 |
| **역할** | 악마 군단의 차원 균열 출입구 |
| **분류** | 금단 구역 (Forbidden Zone) |

---

## 개요

악마 군단이 현실 세계로 침투하는 차원의 균열. 금지된 의식이 끊임없이 이루어지며, 균열을 통해 다양한 악마적 존재들이 출현한다.

---

## 구조

### 균열 상태
| 상태 | 특징 |
|------|------|
| **주 균열** | 대형 악마 통과 가능 |
| **부 균열** | 소형 악마 다수 출현 |
| **봉인 구역** | 부분 봉인, 불안정 |

### 주변 시설
- 악마 소환진
- 피의 제단
- 봉인 파괴 장치

---

## 위험도: 극고

- 현실 왜곡 상시 발생
- 악마 무작위 출현
- 정신 오염 위험

---

## 에피소드 훅

1. **균열 봉쇄** - 지옥문 봉인 미션
2. **악마 쫓기** - 탈출한 악마 추적
3. **금지된 의식** - 의식 저지 작전
4. **역침투** - 균열을 통해 반대편 잠입
