# 심연의 입 (Abyssal Maw)

> *"여긴 땅의 끝이 아니다. 여기서부터는 이야기들이 곧장 떨어져 내리는 곳이다."*

---

## 1. 기본 정보

| 항목 | 내용 |
|------|------|
| **위치** | 흑오벨리스크 남동측 심부, 가장 깊고 오래된 균열이 무너진 구렁텅이 |
| **지형** | 거대한 나선형 함몰 지대. 깊어질수록 중력/빛/시간 왜곡 |
| **역할** | 심연 군단: 지상 출입구 / 망자의 왕국: 최하단 방파제 / 잊힌 자들: 최종 삭제 직전 경계선 |

---

## 2. 구조

### ■ 층위별 구역

| 구역 | 내용 |
|------|------|
| **울부짖는 입구대** | 상단부 절벽. 수없이 떨어진 자들이 올라오려 애쓰는 소리 |
| **나선 절벽길** | 수십·수백 바퀴 도는 좁은 절벽길. 미끄러지면 기억 일부 상실 |
| **속삭임의 갱도** | 중층부 공명 갱도. 심연 속 소리가 언어로 변환 |
| **검은 호수** | 떨어진 이름·기억·감정이 응축된 반액체 심연 물질 |
| **침묵의 턱** | 심연 그 자체로 통하는 문턱. 이 아래는 직접 내려간 사례 거의 없음 |

---

## 3. 세력별 입장

| 세력 | 입장 |
|------|------|
| **심연 군단** | 지상으로 올라오는 메인 출입구. 속삭임/영향력 송신탑 |
| **망자의 왕국** | 영혼이 더 깊이 떨어지지 않도록 막는 최하단 방파제 |
| **잊힌 자들** | 기억·이름이 완전 삭제되기 직전의 마지막 경계선 |
| **봉인 수호단** | 균열 확대 감시. 흑오벨리스크 다음으로 위험한 관측 대상 |

### ■ 공통 금기
- 심연의 입 가장자리에서 **누군가의 이름을 고의로 크게 부르는 행위** 금지
- 그렇게 부른 이름은 그 사람이 없어도 **"심연의 리스트"에 자동 등재**

---

## 4. 전략적 의미

- **심연 군단 보급로/전략거점**: 속삭임을 세계 각지로 퍼뜨리는 송신탑
- **침공/방어 기준선**: 심연이 이 점 이상으로 올라오면 전면전 불가피
- **심연 에너지 수위**가 전술지표 (단순 영토 점령 아님)
