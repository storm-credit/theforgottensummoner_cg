# 5. 퓨리피아 (Purifia)

> *"불은 모든 것을 깨끗하게 한다. 죄도, 병도, 육체까지도."*
> — 화염 정화 부대 지휘관

---

## 기본 정보

| 항목 | 내용 |
|------|------|
| **위치** | 성국 서부, 화산 지대 입구 |
| **역할** | 오염 물질 소각 + 전염병 격리 + 이단 서적 파기 |
| **별명** | "검은 연기의 성", "불타는 쓰레기통" |
| **이미지** | 끊임없이 검은 연기를 내뿜는 거대한 굴뚝들이 솟아 있는 요새. 주변은 재로 뒤덮여 회색빛임 |

### 핵심 시설
- 대소각로 (Great Incinerator)
- 격리 병동 (Isolation Ward)
- 정화의 제단 (Altar of Purification)
- 화산 지열 발전소 (Geothermal Power Plant)

## 지형 및 요새 구조

### 상징적 지형

| 지형 | 설명 |
|------|------|
| **재의 평원 (Ash Plains)** | 요새 주변 수 km는 소각로에서 나온 재가 눈처럼 쌓여 있음. 식물이 자라지 않는 불모지 |
| **용암 해자 (Lava Moat)** | 자연 용암을 끌어와 요새 주변을 흐르게 하여 접근을 차단함 |
| **열기의 벽** | 요새 내부는 항상 40도 이상의 고온을 유지. 병균이 살 수 없는 환경을 인위적으로 조성 |

### 주요 구역

| 구역 | 특징 |
|------|------|
| **소각 구역** | 전국에서 수거된 쓰레기, 시체, 이단 물품을 태우는 곳. 24시간 가동 |
| **격리 구역** | 치료 불가능한 전염병 환자나 저주받은 짐승들을 가두는 곳. 이들이 죽으면 바로 옆 소각장으로 직행 |
| **거주 구역** | 방열복과 마스크를 쓴 병사와 인부들이 사는 곳. 공기가 매우 나쁨 |

## 정치·행정 구조

| 기관 | 역할 |
|------|------|
| **정화 관리국** | 무엇을 태우고 무엇을 남길지 결정하는 권한. 증거 인멸의 중심지 |
| **방역 사령부** | 전염병 발생 시 군대를 동원해 마을을 태워버릴 수 있는 무소불위의 권한 |

### 권력의 현실
> "태우면 그만이다."

- **증거 인멸**: 정치적 암살이나 비리의 증거(시체, 문서)가 이곳에서 "전염병 오염 물품"으로 분류되어 사라짐
- **혐오 시설**: 누구나 필요성은 인정하지만 아무도 가고 싶어 하지 않는 3D 업종. 이곳 근무자들은 천대받음

## 군사 & 치안

| 조직 | 역할 |
|------|------|
| **클린 스위퍼 (Clean Sweepers)** | 화염방사기로 무장한 특수 부대. 오염된 모든 것을 태워버림 |
| **화산 감시대** | 화산 폭발 징후를 감시하고, 용암 몬스터의 침입을 막음 |

### 방어 체계
- **화염 폭풍**: 적이 침입하면 소각로의 배기구를 역류시켜 요새 전체를 불바다로 만듦
- **독가스**: 불완전 연소 가스를 적진에 살포하는 생화학 전술도 사용 (비공식)

---

## 신앙 · 일상 · 분위기

### 신앙
- **불의 정령**: 솔라의 빛보다는 '열기'와 '파괴'를 숭배. 불이 모든 부정을 씻어준다고 믿음
- **결벽증**: 오염에 대한 강박적인 공포가 신앙처럼 자리 잡음

### 일상 풍경
| 시간 | 모습 |
|------|------|
| 아침 | 잿빛 하늘 때문에 해가 잘 보이지 않는 우울한 아침, 마스크를 쓰고 출근하는 인부들 |
| 낮 | 웅웅거리는 소각로 소음, 역겨운 냄새, 끊임없이 실려 오는 시체 마차들 |
| 저녁 | 붉은 용암 빛이 조명을 대신하고, 독한 술로 하루의 메스꺼움을 씻는 병사들 |
| 밤 | 밤새 타오르는 소각로 불빛 때문에 대낮처럼 환함 |

### 요새 정서
> [!note] 분위기
> "죽음과 정화."
> 성국의 가장 더러운 부분을 처리하는 곳. 누군가는 해야 할 일이라는 사명감과 자기혐오가 공존함.

## 순리 & 서사 메모

> [!note] 순리
> 호흡기 질환? → **'정화 필터 마스크'** 보급 필수. 그래도 평균 수명이 짧음
> 연료 공급? → 화산 지열 + 가연성 쓰레기 + 마석

> [!important] 서사 활용
> - **좀비 사태**: 소각 대기 중이던 시체들이 되살아나 요새를 점령. 화염방사기로 통로를 뚫으며 탈출하는 액션
> - **재의 꽃**: 잿더미 속에서 피어난 하얀 꽃 한 송이를 보며 생명의 끈질김을 느끼는 감성적인 장면
