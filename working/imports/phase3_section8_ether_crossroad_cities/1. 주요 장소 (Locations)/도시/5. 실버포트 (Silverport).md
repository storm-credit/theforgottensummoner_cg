# 실버포트 (Silverport)

> *"이곳의 아침은 생선 비린내가 아니라, 버터 구이 냄새로 시작된다."*
> — 미식가 협회 가이드북

---

## 기본 정보

| 항목 | 내용 |
|------|------|
| **위치** | 에테르 대륙 남부 해안, [[2. 포트 넥서스]] 동쪽에 위치 |
| **역할** | 수산물 유통의 중심, 고급 식자재 무역, 휴양지 |
| **별명** | 은빛 항구 (바다에 비친 햇살 때문에), 미식의 천국 |
| **이미지** | 하얀 벽과 푸른 지붕을 가진 지중해풍 건물들, 항구 가득한 요트와 어선 |

### 핵심 시설
- **새벽 어시장 (Dawn Market)**
- **미식가 거리 (Gourmet Street)**
- **은빛 백사장 (Silver Beach)**
- **[[3. 스톰 브레이커 (Storm Breaker)]]** (해상 방어벽)

## 도시 구조

### 상징적 지형

| 지형 | 설명 |
|------|------|
| **은빛 만 (Silver Bay)** | 멸치 떼나 정어리 떼가 들어오면 바다가 은색으로 빛난다고 해서 붙은 이름. |
| **소금 절벽** | 천연 소금이 결정화되어 하얗게 빛나는 절벽. 최고급 소금 산지. |
| **산호 정원** | 얕은 바다에 펼쳐진 아름다운 산호 군락. 관광 명소. |

### 주요 지구

| 지구 | 특징 |
|------|------|
| **항구 지구** | 어선들이 정박하고 경매가 열리는 활기찬 곳. |
| **관광 지구** | 고급 호텔, 레스토랑, 별장들이 모여 있는 휴양지. |
| **가공 지구** | 생선을 염장하거나 통조림으로 만드는 공장. |
| **어부 마을** | 토박이 어부들이 사는 소박하고 낡은 구역. |

## 정치·행정 구조

| 기관 | 역할 |
|------|------|
| **수산업 협동조합** | 어획 쿼터와 가격을 결정하는 실세. |
| **관광청** | 도시 이미지를 관리하고 축제를 기획함. |
| **요리사 길드** | 식당 위생 등급을 매기고 맛집 인증을 함. (권위가 높음) |

### 권력의 현실
> "가장 신선한 생선을 쥐는 자가 권력을 쥔다."

- **[[02. 드레이크 루가]]** 제독(포트 넥서스)의 해군이 이곳의 안전을 보장해 주는 대가로 식량을 징발해 감.
- 해적들도 맛있는 밥을 먹으러 올 때는 무기를 맞기고 얌전히 식사만 하고 가는 불문율이 있음.

## 군사 & 치안

| 조직 | 역할 |
|------|------|
| **해안 경비대** | **[[3. 스톰 브레이커]]** 소속 파견대. 해적과 해수 방어. |
| **시장 관리인** | 어시장의 질서를 유지하는 덩치들. 생선 칼을 잘 씀. |

### 특수 상황
- **해수(Sea Monster) 출몰:** 가끔 거대 문어나 상어가 해수욕장에 나타나 비상이 걸림.
- **식중독 테러:** 라이벌 식당이나 도시를 음해하려는 세력이 독을 푸는 것을 가장 경계함.

## 신앙 · 일상 · 분위기

### 신앙
- **요리의 신** & **바다의 신** 숭배.
- "맛없는 음식은 신성모독이다."

### 일상 풍경
| 시간 | 모습 |
|------|------|
| **새벽** | 만선의 깃발을 단 배들이 들어오고, 경매사의 종소리가 울림. |
| **낮** | 관광객들이 해변을 거닐고, 식당마다 맛있는 냄새가 진동함. |
| **밤** | 해변에서 불꽃놀이와 바비큐 파티가 열림. |

### 도시 정서
> "여유롭고 풍요로움. 먹고 마시고 즐기는 쾌락주의."

## 소환 · 균열 관련

### 공식 입장
- **"맛있는 마수를 소환할 수 있나?"**
- 식용 가능한 몬스터(크라켄 다리, 드래곤 꼬리 등)를 조달하는 소환사는 환영.

### [[01-15. 인물 백과 (Character Archive)/0. 주인공 일행 (Main Characters)/에반 무영 라크라시스 (Evan)]] 개입 포인트
- **전설의 식재료:** 심해 깊은 곳에 사는 전설의 물고기를 잡아달라는 요리사의 의뢰.
- **해적 요리사:** 은퇴한 전설의 해적이 운영하는 식당에서 단서를 얻음.
- **해수 퇴치:** 해수욕장에 난입한 몬스터를 처리하고 '영웅' 대접(공짜 식사권)을 받음.

## 순리 & 서사 메모

> [!note] 순리
> 거친 항구(포트 넥서스) 옆에 위치한 휴양 및 보급 도시.
> 전투에 지친 주인공 일행이 잠시 쉬어가는 **'힐링 에피소드'** 장소.

> [!important] 서사 활용
> - **'던전밥'** 스타일의 미식 판타지.
> - 평화로운 표면 아래 숨겨진 해적들의 밀수 루트 (생선 상자에 마약 숨기기).
> - **[[3. 스톰 브레이커]]**와의 협력 방어전.
