# 포트 넥서스 (Port Nexus)

> *"바다는 묻지 않아. 네가 왕인지 거지인지. 그저 삼킬 뿐이지."*
> — 항구 입구의 낡은 비석

---

## 기본 정보

| 항목 | 내용 |
|------|------|
| **위치** | 에테르 대륙 남서부 끝자락, 거대한 만(Bay) 안쪽 |
| **역할** | 대륙 간 무역의 **최대 관문**, 연합 해군 본부, 밀수 허브 |
| **별명** | 해상 관문 (The Sea Gate), 이중 도시 (낮과 밤이 다름) |
| **이미지** | 숲처럼 빽빽하게 솟은 돛대들, 짠내와 생선 비린내, 그리고 화약 냄새 |

### 핵심 시설
- **연합 해군 사령부 (Navy HQ)**
- **아이언 독 (Iron Dock)** - 군함 전용 조선소
- **검은 닻 거리 (Black Anchor Street)** - 유흥가 및 암시장
- **[[3. 스톰 브레이커 (Storm Breaker)]]** - 외곽 방어 요새

## 도시 구조

### 상징적 지형

| 지형 | 설명 |
|------|------|
| **제1부두 (Royal Pier)** | 대형 무역선과 VIP들이 이용하는 고급 부두. 세관 검사가 까다로움. |
| **해무의 언덕** | 항구 전체가 내려다보이는 언덕. 제독 관저가 위치함. |
| **난파선 무덤** | 항구 외곽의 암초 지대. 버려진 배들이 엉겨 붙어 기괴한 모습을 함. |

### 주요 지구

| 지구 | 특징 |
|------|------|
| **무역 지구** | 합법적인 창고와 무역 회사들이 위치. 낮에는 매우 분주함. |
| **군사 지구** | 일반인 출입 금지. 거대한 마도함선들이 정박해 있음. |
| **환락 지구** | 선원들을 위한 술집, 도박장, 숙박업소가 밀집. 밤의 주인은 해적임. |
| **빈민가 (쥐구멍)** | 밀항자와 도망자들이 숨어 사는 부두 밑 찜질방 같은 공간. |

## 정치·행정 구조

| 기관 | 역할 |
|------|------|
| **해군 사령부** | **[[02. 드레이크 루가]]** 제독이 도시를 군사적으로 통제함. |
| **항만청** | 입출항 관리 및 세금 징수. 뇌물이 통하는 곳으로 유명함. |
| **그림자 조합** | **[[09. 셀레나 와일드웨이브]]**가 이끄는 뒷골목 조직. 해적들과의 중재 담당. |

### 권력의 현실
> "낮에는 제독의 법이, 밤에는 해적의 칼이 지배한다."

- 드레이크 제독의 **군사 독재**에 가깝지만, 경제를 위해 상인들의 자치권을 어느 정도 인정함.
- 해적 연합과의 '불가침 [[01-37. 소환 백과/05. 소환 7대 조약 및 차원 체계/7조약]]'이 암묵적으로 유지되고 있음 (상납금을 내면 공격하지 않음).

## 군사 & 치안

| 조직 | 역할 |
|------|------|
| **연합 해군 (Union Navy)** | 정규군. 해상 전투력은 대륙 최강. |
| **헌병대** | 도시 내부 치안 담당. 해적들과 사이가 매우 나쁨. |
| **바다의 아이들** | 셀레나의 정보원들. 구두닦이, 짐꾼 등으로 위장함. |

### 특수 상황
- **밀수 성행:** 세관의 눈을 피한 금지 물품(마약, 불법 마도구) 거래가 활발함.
- **해적 침공:** 가끔 협상이 결렬되면 해적선단이 항구 앞바다까지 와서 무력 시위를 함.

## 신앙 · 일상 · 분위기

### 신앙
- **포세이돈 (바다의 신)** 및 각종 토속 해양신 숭배.
- 출항 전 제물(술이나 동전)을 바다에 던지는 풍습이 있음.

### 일상 풍경
| 시간 | 모습 |
|------|------|
| **새벽** | 밤새 조업을 마친 어선들의 귀항과 경매. |
| **낮** | 짐을 싣고 내리는 크레인의 소음, 상인들의 고함. |
| **밤** | 술 취한 선원들의 싸움질, 뒷골목에서 은밀하게 오가는 검은 돈. |

### 도시 정서
> "거칠고 야만적이지만, 누구보다 자유로운 영혼들의 도시."

## 소환 · 균열 관련

### 공식 입장
- **"물에 젖어도 쓸 수 있는 소환수만 오라."**
- 물 속성, 바람 속성 소환사는 우대 채용 (항해에 도움).
- 불 속성 소환사는 목선(木船) 화재 위험 때문에 기피 대상.

### 실제 상황
- **심해의 괴수:** 항구 깊은 곳에서 가끔 '크라켄' 같은 심해 마수가 튀어나옴.
- **수중 [[01-37. 소환 백과/01-13. 소환술 (Summoning System)]]:** 해적들은 물속에서 암살할 수 있는 소환수를 비밀리에 육성함.

### [[01-15. 인물 백과 (Character Archive)/0. 주인공 일행 (Main Characters)/에반 무영 라크라시스 (Evan)]] 개입 포인트
- **밀항 작전:** 수배 중에 검문을 피해 다른 대륙으로 가는 배에 탑승해야 함.
- **해전 지휘:** 해적들의 대규모 침공 시, 소환수를 이용해 해군과 협력하여 방어.
- **난파선 인양:** '난파선 무덤'에 가라앉은 고대 보물선을 소환수의 힘으로 인양.
