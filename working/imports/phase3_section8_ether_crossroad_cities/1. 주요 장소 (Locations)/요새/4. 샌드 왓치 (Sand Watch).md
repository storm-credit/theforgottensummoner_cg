# 샌드 왓치 (Sand Watch)

> *"모래바람 속에서는 적보다 목마름이 먼저 너를 죽일 것이다."*
> — 사막 수비대 격언

---

## 기본 정보

| 항목 | 내용 |
|------|------|
| **위치** | 에테르 대륙 남동부 사막 지대 경계, **[[8. 포트 셰어]]** 서쪽 |
| **역할** | 사막 몬스터 및 도적단 감시, 오아시스 보호, 동부 무역로 수호 |
| **별명** | 사막의 눈, 신기루 요새, 전갈 둥지 |
| **이미지** | 황토색 벽돌로 지어진 돔 형태의 요새, 끝없는 모래언덕 위에 홀로 서 있음 |

### 핵심 시설
- **대수조 (Grand Cistern)** - 요새 지하의 거대 물 저장고. 생명줄.
- **전갈 감시탑** - 사막 전역을 감시하는 높은 탑.
- **지하 격납고** - 샌드 스키머(모래 위를 달리는 배) 보관소.
- **냉각소** - 마법으로 실내 온도를 낮추는 휴식 공간.

## 요새 구조

### 상징적 지형

| 지형 | 설명 |
|------|------|
| **붉은 사막** | 요새 앞쪽으로 펼쳐진 붉은 모래 사막. 낮에는 50도까지 올라감. |
| **선인장 숲** | 요새 주변에 심어진 거대 선인장 군락. 자연 방어벽이자 식수원. |
| **유사 지대** | 빠지면 헤어 나올 수 없는 모래 늪. 지도를 모르면 죽음. |

### 주요 구역

| 지구 | 특징 |
|------|------|
| **지상 구역** | 감시탑과 성벽 등 방어 시설. 낮에는 너무 더워 최소 병력만 배치. |
| **지하 구역** | 대부분의 생활 공간과 훈련소. 시원하고 습도가 유지됨. |
| **오아시스 구역** | 요새 안뜰에 있는 작은 샘. 성역으로 취급됨. |
| **무역로 관문** | 사막을 건너온 상단들이 쉬어가는 검문소. |

### 권력의 현실
> "물 한 모금이 금화 한 자루보다 귀하다."

- **[[1. 골든마르크]]**의 자본으로 지어졌지만, 운영은 현지 방식(부족 전통)을 따름.
- **[[11. 마리안 카르도]]**의 운송 길드가 이곳을 거쳐 사막 무역을 독점하고 있음.
- **[[8. 포트 셰어]]**와 공생 관계 (물과 사치품의 교환).

## 소환 · 균열 관련

### 공식 입장
- **"물 소비가 적은 소환수만 허용."**
- 물을 많이 먹거나(슬라임 등), 열에 약한 소환수는 민폐.
- 불 속성, 땅 속성 소환사는 매우 환영받음.

### [[01-15. 인물 백과 (Character Archive)/0. 주인공 일행 (Main Characters)/에반 무영 라크라시스 (Evan)]] 개입 포인트
- **수원 확보:** 오아시스가 말라가는 원인을 찾고, 지하 수맥을 뚫어줌.
- **도적 소탕:** 사막 도적단 '검은 전갈'의 본거지를 찾아내 섬멸.
- **고대 유적:** 모래폭풍이 지나간 뒤 드러난 고대 피라미드를 탐사.
