# 최초의 무덤 (First Tomb)

> *"여기에 첫 번째 이름이 새겨졌고, 첫 번째 죽음이 기록되었다."*

---

## 1. 기본 정보

| 항목 | 내용 |
|------|------|
| **위치** | 망자의 평원 최심부, 사자의 수도보다 더 깊은 지하 |
| **외관** | 단일 석관 하나만 있는 거대한 빈 공간 |
| **역할** | 망자의 왕국 기원지 / 최초의 망자왕 안치소 / 최고 성지 |

---

## 2. 성지의 의미

### ■ 창건 신화
- **최초의 망자왕**이 스스로 죽음을 선택하고 이곳에 묻힘
- 그의 마지막 말: *"이름을 기억하는 한, 죽음은 끝이 아니다"*
- 이 유언이 망자의 왕국 건국 이념

### ■ 왕위 계승
- 새 망자의 왕은 이곳에서 **첫 번째 이름을 받음**
- 실제로는 최초의 왕에게 "승인"받는 의식

---

## 3. 구조

| 구역 | 설명 |
|------|------|
| **하강 계단** | 999계단. 각 계단에 역대 왕의 이름 |
| **망각 회랑** | 이름 없이 묻힌 자들의 위패 |
| **석관의 방** | 최초의 왕 석관. 열어본 자 없음 |
| **반향실** | 최초의 왕에게 질문하면 대답이 돌아옴 (가끔) |

---

## 4. 접근 제한

- 왕과 기억 귀족 최상위만 출입 허용
- 외부인 접근 = 즉시 망자의 저주 (이름 일부 소멸)
