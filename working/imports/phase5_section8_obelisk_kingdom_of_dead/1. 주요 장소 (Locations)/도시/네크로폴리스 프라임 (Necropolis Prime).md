# 사자의 수도 (Necropolis Prime)

> *"이 도시에선, 태어난 날보다 쓰여진 날과 묻힌 날이 더 중요하다."*

---

## 1. 기본 정보

| 항목 | 내용 |
|------|------|
| **위치** | 흑오벨리스크에서 비켜선 반경 안, 비교적 안정된 지반 |
| **지형** | 계단식 무덤 평원 위에 얹힌 수직 도시 |
| **역할** | 망자의 왕국 수도 / 행정·의례·기억 관리 중심 |
| **통치자** | [[망자의 왕국 (Kingdom of the Dead)|망자의 왕 (King of the Dead)]] |

---

## 2. 도시 구조

### ■ 지상 구역

| 구역 | 설명 |
|------|------|
| **왕관 피라미드 지구** | 망자의 왕이 머무는 최중앙·최상층 피라미드. 청백색 의식의 불꽃 |
| **기억 궁정 지구** | 기억 귀족들의 탑·궁전. 이름과 연대기가 외벽에 부조로 새겨짐 |
| **영혼 행렬로** | 동심원 형태의 의식용 도로. 바닥에 과거 행렬의 이름·날짜 새겨짐 |
| **무덤·탑 묘지 구역** | 다층 납골탑, 고위층 묘당, 군단 전사 집단 묘지 |
| **산 자의 거주지** | 묘지 관리인, 의식 집행자, 외부 사절단 숙소 |

### ■ 지하 구조

| 구역 | 설명 |
|------|------|
| **대카타콤 그리드** | 도시 전체 아래 수평형 묘지망. 시대/전쟁/왕조별 구분 |
| **왕의 심연 궁정** | 가장 깊은 층. 왕의 옥좌, 이름 서고, 기억 귀족 회의실 |

## 3. 정치·권력 구조

### ■ 지배 체계
```
망자의 왕 (The Thanatarch)
    ↓

    ↓
묘역 감독관 → 하위 기록관·관리인
```

### ■ 내부 파벌

| 파벌 | 노선 |
|------|------|
| **기억 판매파** | 기억을 팔아 외부와 거래하는 실용주의 |
| **보존파** | 기억을 절대 외부에 넘기지 않는 원칙주의 |
| **무명파** | 이름을 버리고 도시 자체의 의지에 동화되려는 집단 |

## 4. 군사적 의미

- **죽어도 끝나지 않는 동원력**: 이름이 보관된 자는 언제든 재소환 가능
- **함락 불가능한 수도**: 지상이 무너져도 카타콤과 기록이 남아 전쟁 지속
- **타 세력 입장**:
  - 심연 군단: 질서 있는 죽음을 심연으로 통째로 흡수 욕망
  - 잊힌 자들: 전쟁/시대의 진실을 통째로 복원할 수 있는 정보원
