# 망자의 성채 (Citadel of the Damned)

> *"살아있는 자는 이곳의 손님이 아니다. 먹잇감이다."*

| 항목 | 내용 |
|------|------|
| **위치** | 오벨리스크 대륙 서부, 망자의 평원 중심부 |
| **역할** | 언데드 군단 수도, 저주받은 왕의 거처 |
| **분류** | 수도 (Capital) |

---

## 개요

사망한 병사들이 끝없이 부활하는 저주받은 성채. 저주받은 왕이 통치하며, 망자의 평원 전체를 지배한다.

---

## 구조

### 주요 구역
| 구역 | 특징 |
|------|------|
| **저주왕의 옥좌** | 저주받은 왕 거처 |
| **부활의 성당** | 대규모 부활 의식 |
| **망령 귀족관** | 언데드 귀족 거주 |
| **저주의 탑** | 저주 마법 연구 |
| **영혼 감옥** | 포획된 영혼 보관 |

### 저주의 효과
- 죽은 자 자동 부활
- 생명력 흡수
- 시간 정지 영역

---

## 에피소드 훅

1. **왕의 저주** - 저주받은 왕 제거 미션
2. **영혼 해방** - 감금된 영혼 구출
3. **저주 해제** - 성채의 저주 해제 방법 탐색
4. **잠입 정찰** - 적 거점 정보 수집
