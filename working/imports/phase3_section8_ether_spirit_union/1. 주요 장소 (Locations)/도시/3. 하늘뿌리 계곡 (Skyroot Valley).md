# 3. 하늘뿌리 계곡 (Skyroot Valley)

> *"나무 뿌리가 하늘을 향해 자란다. 여기서는 위와 아래가 바람에 따라 달라지기 때문이다."*

---

## 기본 정보

| 항목 | 내용 |
|------|------|
| **위치** | 에테르 대륙 동부 산악 지대, 구름 사이 고원 계곡 |
| **역할** | 산지 성역, 바람·대지 정령 본거지 |
| **특징** | 거꾸로 자라는 듯한 나무들과 부유하는 바위 |
| **별칭** | "바람의 정원", "거꾸로 숲", "구름 위 마을" |

---

## 지형 & 현상

### 물리적 특성
| 항목 | 설명 |
|------|------|
| **고도** | 해발 3,000m+ |
| **기후** | 서늘하고 바람이 강함 |
| **특이 현상** | 부유 바위, 역방향 수목 |
| **접근성** | 특정 시간대만 안개가 걷혀 접근 가능 |

### 거꾸로 나무 (Skyroot Trees)
- 뿌리가 하늘을 향해 뻗어 있는 것처럼 보이는 고대 나무
- 실제로는 바람 정령의 축복으로 성장 방향 왜곡
- 나뭇잎이 구름을 빗물처럼 걸러냄

### 부유 바위 (Floating Stones)
- 계곡 곳곳에 떠 있는 거대 암석
- 대지 정령과 바람 정령의 균형으로 부유
- 일부는 거주지/의식 장소로 사용

---

## 핵심 구역

### 1. 바람의 제단 (Altar of Winds)
| 항목 | 내용 |
|------|------|
| **위치** | 계곡 중앙의 가장 큰 부유 바위 위 |
| **특징** | 사방에서 바람이 소용돌이치는 신성한 장소 |
| **기능** | 바람 정령 계약 의식, 예언 해석 |

### 2. 뿌리 마을 (Root Village)
- 거꾸로 나무들 사이에 형성된 거주 구역
- 나무 줄기와 부유 바위를 연결하는 현수교
- 엘프와 인간 드루이드 혼합 거주

### 3. 구름 관측소 (Cloud Observatory)
- 가장 높은 부유 바위 위
- 날씨 예측, 균열 징조 감지
- 별 아래 부족과 정보 교환

### 4. 대지의 심장 (Heart of Earth)
- 계곡 가장 깊은 곳
- 대지 정령이 가장 강하게 느껴지는 동굴
- 지진, 산사태 조기 감지

## 사회 구조

### 자치 체계
| 직위 | 역할 |
|------|------|
| **바람의 수호자** | 계곡 전체 관리, 바람 정령 계약 최고위 |
| **뿌리 장로회** | 마을 운영, 외부 교류 결정 |
| **구름 예언자** | 기후, 재앙, 균열 징조 해석 |

### 정령 관계
- **바람 정령(실프)**: 계곡의 주요 파트너, 이동·통신 보조
- **대지 정령(노움)**: 부유 바위 안정 유지, 동굴 보호
