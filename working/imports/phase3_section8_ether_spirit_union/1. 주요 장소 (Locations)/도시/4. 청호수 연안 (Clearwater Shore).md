# 4. 청호수 연안 (Clearwater Shore)

> *"호수가 맑으면 정령도 맑다. 호수가 흐려지면 우리 모두의 마음이 흐려진다."*

---

## 기본 정보

| 항목 | 내용 |
|------|------|
| **위치** | 에테르 대륙 서부, 거대 담수호 주변 |
| **역할** | 물 정령 성역, 호수 마을 연합, 치유·정화 중심지 |
| **특징** | 수면이 거울처럼 맑아 하늘이 완벽하게 비침 |
| **별칭** | "거울 호수", "정령의 눈", "맑은 물의 땅" |

---

## 지형 & 현상

### 물리적 특성
| 항목 | 설명 |
|------|------|
| **호수 크기** | 직경 약 30km |
| **수심** | 중앙 최대 200m+ |
| **수질** | 거의 완벽한 투명도 |
| **온도** | 사계절 15°C 내외 유지 |

### 거울 수면 (Mirror Surface)
- 바람이 없을 때 하늘이 완벽하게 반사
- 물 정령의 축복으로 불순물 자동 정화
- "호수를 들여다보면 진짜 자신이 보인다"

### 수중 정원 (Underwater Garden)
- 호수 바닥에 발광 식물과 수중 숲
- 물 정령들의 거주 공간
- 특별 의식 때만 인간 잠수 허용

---

## 핵심 구역

### 1. 부유 마을 (Floating Villages)
| 항목 | 내용 |
|------|------|
| **구조** | 호수 위에 떠 있는 목조 플랫폼 마을 |
| **연결** | 다리와 보트로 서로 연결 |
| **거주민** | 어부, 드루이드, 정령술사 |

### 2. 달빛 제단 (Moonlight Altar)
- 호수 중앙의 작은 섬
- 보름달 밤에만 의식 진행
- 물 정령과의 최고 계약 장소

### 3. 치유의 샘 (Healing Springs)
- 호수 주변 여러 온천
- 물 정령의 축복이 담긴 치유수
- 부상자, 저주받은 자 치료

### 4. 물결의 학교 (Ripple Academy)
- 물 정령술사 양성 기관
- 정화 의식, 치유 마법, 물 조종 교육
- 마법협회와 비공식 교류

## 사회 구조

### 마을 연합
| 직위 | 역할 |
|------|------|
| **호수 수호자** | 청호수 연안 전체 대표, 달빛 제단 관리 |
| **물결 장로회** | 각 부유 마을 대표 연합 |
| **깊은 물의 [[01-20. 직업 & 클래스 (Classes & Jobs)/직업 도감/08. 사제 (Cleric)]]** | 수중 정원 관리, 물 정령 최고위 계약자 |

### 생활 양식
- 어업과 수초 채집 중심
- 금속 사용 최소화 (물 정령 선호)
- 노래로 물고기를 부르는 전통

## 경제 & 생산

### 주요 자원
| 자원 | 용도 |
|------|------|
| **청수 진주** | 마법 재료, 정화 효과 |

| **발광 수초** | 조명, 마법 시약 |
| **호수 물고기** | 특수 효능 (체력 회복) |

### 교역
- 치유수와 진주는 각 대륙에서 수요 높음
- 자유도시연합과 정기 교역
- 마나 재료로 마법협회와 거래

## 방어 & 전략

### 수상 방어
- 호수 자체가 천연 방어선
- 물 정령이 침입 선박 침몰 가능
- 수중 경로로 비상 대피

### 상주 전력
| 유닛 | 규모 |
|------|------|
| **파도 [[01-20. 직업 & 클래스 (Classes & Jobs)/직업 도감/06. 기사 (Knight)]]단** | 40명, 수상/수중 전투 전문 |
| **물결 드루이드** | 25명, 정령 소환/치유 |
| **호수 순찰대** | 50명, 연안 감시 |

### 위기 시
- 물 정령이 **호수 전체를 폭풍으로 변환** 가능
- 최후 수단: 호수 아래로 마을 전체 대피
