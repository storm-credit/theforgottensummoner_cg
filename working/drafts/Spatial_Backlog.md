# Spatial Backlog

## Rule

이 문서는 지도 축을 `점진적으로` 붙이기 위한 backlog다.

- 지금 당장 완성하지 않는다.
- 메인 진행을 방해하지 않는다.
- 각 단계에서 필요한 최소 지도만 추가한다.

## Layer 1. World / Continent

- 세계지도 초안
- `1~5 대륙` 상대 위치
- 핵심 항로
- 대륙 간 병목
- 범대륙 축과 분리된 핵심 무대

## Layer 2. Regional

- 각 대륙 핵심 권역
- 변경 지대
- 가문 권역
- 부족 활동권
- 길드 / 상단 거점

## Layer 3. Urban

- 수도
- 자유도시
- 항구 도시
- 시장 / 귀족가 / 빈민가 / 신전가

## Layer 4. Landmark

- 왕성
- 마탑
- 신전
- 길드 본부
- 금고
- 연구소
- 요새

## Layer 5. Interior

- 성 내부
- 신전 내부
- 연구소 내부
- 금고 내부
- 비밀 통로
- 잠입 경로

## Immediate Backlog

1. 에테르 대륙 권역 메모
2. 에테르 대륙 핵심 세력 본거지 후보 수집
3. 자유도시연합 도시 기능 구역 메모
4. 성국 / 왕국연합 / 마법협회의 대표 랜드마크 후보 수집
5. 자유도시연합 `항구-상단-용병-암시장` 동선 시드 정리
