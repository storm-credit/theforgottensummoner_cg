# Crossroad Cities Map Seed

이 문서는 `자유도시연합`의 도시 지도를 나중에 실제로 그릴 때
바로 사용할 수 있는 1차 공간 시드다.

아직 완성 지도는 아니고,
`누가 어디서 움직이는가`를 먼저 잡기 위한 메모다.

## Core Urban Spine

### 1. 골든마르크

- role:
  - 비공식 수도
  - 공식 상업권과 외교권의 중심
- likely anchors:
  - `세실리아 메르카토르`
  - `페어딜러 상단`
  - `레이나 브라이트헤븐`
  - `이사벨 카르도`
- map use:
  - 상단 회관
  - 공식 경매장
  - 외교 회담장
  - 운송 길드 허브

### 2. 포트 넥서스

- role:
  - 해양 관문
  - 밀수와 합법 무역이 충돌하는 항만 도시
- likely anchors:
  - `모이라 와일드웨이브`
  - `밤의 무역상`
  - `대륙 무역 연맹`
- map use:
  - 제3 부두
  - 비밀 창고
  - 주점 `검은 고양이`
  - 세관/검문소

### 3. 머시너리 게이트

- role:
  - 자유도시 용병 연맹 본부 도시권
  - 계약과 배신이 동시에 기록되는 실무 허브
- likely anchors:
  - `자유도시 용병 연맹`
  - `길드 마스터 계열`
  - `중재 위원회 심판관 계열`
  - `블랙리스트 기록관 계열`
- map use:
  - 용병 길드 본부
  - 의뢰 게시 전당
  - 명예의 전당
  - 훈련소
  - 블랙리스트 보드 구역

## City Function Split

자유도시연합 지도는 정치 구획보다
`기능 구획`으로 먼저 나누는 게 맞다.

- `합법 상업 구역`
- `비공식 경매 / 암시장 구역`
- `항만 / 물류 구역`
- `용병 계약 구역`
- `외교 / 중재 구역`
- `하수구 / 지하 운송 구역`

## Human Layer

이 도시 지도의 장점은
공간과 사람이 바로 맞물린다는 점이다.

- `세실리아 메르카토르`
  - 공식 외교 / 협상 / 상업권
- `레이나 브라이트헤븐`
  - 공식 경매 / 귀족 연결 / 비공식 외교 채널
- `이사벨 카르도`
  - 물류 / 운송 / 지하 운송 협력선
- `모이라 와일드웨이브`
  - 밀수 / 해적-내륙 연결 / 밤의 접선망
- `다리안 벨가르드`
  - 계약 / 용병 질서 / 명예의 전당
- `벤투라 모레티`
  - 익명 거래 / 신원 세탁 / 실버포트형 지하 중개
- `도미닉 발사노`
  - 폭력 채권 회수 / 채무 감금 시설
- `로렌조 카르미네`
  - 노예 유통 / 생체 거래 / 검은 시장

## Interior Candidates

나중에 내부 맵으로 가장 먼저 뽑을 후보:

1. `페어딜러 상단 회관`
2. `공식 경매장`
3. `제3 부두 밀수 창고`
4. `주점 검은 고양이`
5. `머시너리 게이트 길드 본부`
6. `실버포트형 비밀 거래장`
7. `채무자 감금 시설`
8. `노예 수용소 / 검은 시장`

## Conductor Note

자유도시연합은
`국가 수도 지도`처럼 그리기보다
`거래와 접선의 도시망`으로 그리는 편이 더 맞다.
