# Continents Framework

이 문서는 `1~5 대륙`의 뼈대를 먼저 고정하기 위한 작업 시트다.

## 1. 에테르 대륙 (Ether Continent)

- 정체성:
- 인간 군상:
- 대표 갈등:
- 대표 세력군:
- 핵심 가문층: 성국 / 왕국연합 축이 강하고, 인물 문서와 세력 문서 모두 가문 기반을 강하게 뒷받침함
- 핵심 부족층: 대륙 공통 축으로는 약하지만 정령연합 내부에서는 부족층이 강하게 보임
- 핵심 길드층: 자유도시연합, 국제 길드 연합, 상단, 카르텔, 정보단, 영약회 축이 강하게 보임

## Rule

- 여기서는 세부 설정보다 뼈대를 먼저 잡는다.
- 각 대륙에서 `가문 / 부족 / 길드`가 얼마나 필요한지 빈칸부터 먼저 본다.
- `범대륙` 파트는 이 문서에서 다루지 않는다.
