# Setting Book Chapter 7 Draft: Spatial Logic, Maps, and Travel Progression

## Chapter Purpose

This chapter defines how the setting book should turn scattered place names into usable geography.

The map should not start as decoration. A good map tells the reader where power sits, where trade moves, where danger gathers, where rumors travel, and why characters must cross uncomfortable borders.

The setting should build maps in layers rather than rushing into a beautiful but unsupported world image.

## Map Progression Order

Use this order when building map material:

1. world axis
2. continent axis
3. regional axis
4. city axis
5. landmark axis
6. building axis
7. interior routes

Do not build city maps before continent structure is stable. Do not build dungeon interiors without knowing who uses the place, who avoids it, and what larger route it belongs to.

## Place Function First

Every place entry should be defined by the scene function it enables.

Core questions:

- Who lives here?
- Who wants to come here?
- Who is expelled or unwelcome here?
- What resource, route, memory, danger, or law makes this place necessary?
- Which faction benefits from controlling it?
- Which faction pretends not to control it?
- What kind of conflict becomes possible only here?
- What changes when a character crosses into or out of it?

Pretty landmarks are not enough. A place earns setting-book space when it changes movement, politics, memory, trade, faith, danger, or relationship.

## Place Function Types

Use these tags when sorting locations.

| Function Type | Meaning |
| --- | --- |
| threshold | Border, gate, crossing, port, pass, bridge, or entry point where status changes. |
| market | Place where goods, rumors, favors, contracts, and debts circulate. |
| memory site | Ruin, battlefield, grave, shrine, seal site, or old disaster location. |
| workshop | Place of craft, research, forging, magical production, or repair. |
| sanctuary | Safe haven, temple, protected village, refuge, or neutral shelter. |
| underworld node | Smuggling point, black market, sewer route, secret warehouse, hidden auction, or criminal contact point. |
| command seat | Palace, guild hall, fortress, order chapter, council chamber, or faction headquarters. |
| trial ground | Dungeon, cursed forest, monster territory, test site, arena, or forbidden zone. |
| route hub | Junction where roads, sea lanes, pilgrim paths, or arcane routes connect. |

## Travel Route Logic

Routes should have ownership, cost, danger, and rumor flow.

Route questions:

- Who controls this route?
- Who pays to use it?
- Who cannot use it openly?
- What blocks it during war, winter, storms, plague, monster migration, piracy, or political tension?
- Who knows the unofficial version of the route?
- Are goods, refugees, letters, relics, fugitives, or rumors moving through it?
- Is it recorded by officials or preserved by smugglers, pilgrims, spirits, or old maps?

Route types:

- royal road
- merchant road
- pilgrim road
- sea lane
- underpath
- arcane route
- supply route
- courier line
- refugee path

## Ether Continent Working Spatial Anchors

The Ether Continent currently has the richest immediate mapping potential.

### Saint Haven

Saint Haven can operate as a sanctuary, command seat, faith-political center, and threshold into holy or lawful authority.

Current anchor candidates:

- Solaris

These should be sorted by whether they are city districts, institutions, landmarks, or symbolic centers.

### Allied Kingdoms

The Allied Kingdoms need a political and military geography that explains cooperation, rivalry, border defense, and noble houses.

Current anchor candidates:

- Adenbrook

These names should become more than dots on a map. Each should eventually answer what it protects, trades, remembers, or fears.

### Arcane Society

Current anchor candidates:

### Spirit Union

The Spirit Union should be mapped through forest settlement logic rather than conventional capital-state logic.

Current anchor candidates:

- Lumira
- Sleeping Spirit Forest
- forest settlements
- mobile camps

Map logic should include seasonal movement, sacred groves, spirit routes, hidden paths, and relations with beastfolk, elves, fairies, and spiritized individuals.

## Crossroad Cities Mapping Principle

The Crossroad Cities need two overlapping maps:

- visible map: legal commerce, diplomacy, port authority, mercenary contracts, public roads
- hidden map: black market auctions, sewer paths, smuggler routes, secret warehouses, unofficial messengers

Important build order:

1. Goldenmark as official commerce and diplomacy center
2. Port Nexus as sea gateway where smuggling and legal trade collide
3. Mercenary Gate as contract and armed-neutrality hub
4. Silverport-style secret trade space if supported by source evidence
5. underground routes linking warehouses, sewers, and unofficial contacts

This creates a place where merchants, nobles, mercenaries, smugglers, relic hunters, and fugitives can all plausibly cross paths.

## Chapter Lock

This chapter locks spatial handling for the current setting-book recomposition pass:

- Build geography by function before visual polish.
- Map world, continent, region, city, landmark, building, and interior routes in order.
- Do not create unsupported city maps before faction and route logic stabilizes.
- Treat Crossroad Cities as both visible legal network and hidden shadow network.
- Make routes carry goods, rumors, danger, and political pressure.
- Use maps to support story movement, not just world decoration.
