# Continental Adequacy Map

이 문서는 `1~5 대륙`이 인간 서사 중심 세계로 기능하기 위해
`가문 / 부족 / 길드` 층이 얼마나 필요한지, 어디가 비어 있는지 먼저 표시하는 점검 시트다.

## Status Labels

- `enough`
- `thin`
- `missing`
- `unknown`

## 3. 프로스트 대륙 (Frost Continent)

- 가문층 상태: `thin`
- 부족층 상태: `enough`
- 길드층 상태: `enough`
- 현재 판단: 프로스트는 프로스트본 연합 하나만으로도 부족 체계가 강하게 선다. 동시에 `아이스블러드 / 울프스피릿 / 스톰베인 / 프리즌윈드` 같은 클랜형 상층과 상인 협회, 방어 협회, 거래 독점 조직이 확인되어 가문/길드도 보수적으로 끌어올릴 수 있다.

## Audit Rule

- 국가와 대형 세력만 보지 않는다.
- 인물 출신, 계승, 복수, 생계, 혼인, 직능, 유통이 실제로 서는 중간 층을 본다.
- `범대륙` 파트는 여기서 평가하지 않는다.

## Immediate Use

1. `Continents_Framework.md`와 같이 본다.
2. 각 대륙에 어떤 중간 사회 단위가 비었는지 먼저 표시한다.
3. 그 다음에 `8. 세력 아카이브`에서 실제 후보를 찾는다.
4. 부족한 층은 새 문서 작성보다 먼저 `결손 유형`으로 기록한다.
5. 현재 배치 기준 첫 실감사 대상은 `에테르 대륙`이다.
