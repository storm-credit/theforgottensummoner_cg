# Phase 5 Findings

이 문서는 `오벨리스크 대륙` 5차 import를 읽고,
`가문 / 부족 / 길드`, 인간 서사 결, 공간축을 보수적으로 정리한 소견서다.

## Main Conclusion

오벨리스크는 `초월/괴물 대륙`으로만 읽으면 손해가 크다.

- 전통 귀족층은 약하다.
- 부족 / 씨족 공동체 층은 거의 보이지 않는다.
- 대신 `기억 귀족`, `기억 지기`, `파벌 대표`, `원로 군관`, `망명 네트워크 통치자` 같은
  비정통 중간 엘리트층이 분명히 존재한다.
- 경제 구조도 약하지 않다.
  기억 거래, 유물 거래, 보급항, 봉인 유지 비용, 저당 경제, 암시장이 실제로 돌아간다.

즉 오벨리스크는 `문명이 무너진 빈 땅`이 아니라,
`붕괴 이후에도 살아남은 인간 조직이 기괴한 방식으로 재편된 대륙`으로 읽는 게 맞다.

## 1. House Layer

- 상태: `thin`
- 판단:
  - 혈통 귀족 중심 대륙은 아니다.
  - 하지만 `망자의 왕국`의 기억 귀족, `잊힌 자들의 연합`의 가문 폴더, 각 세력의 실권 엘리트가
    전통 귀족을 대체하는 중간 지배층 역할을 한다.

- `working/imports/phase5_section8_obelisk_kingdom_of_dead/망자의 왕국 (Kingdom of the Dead).md`

## 2. Tribe Layer

- 상태: `missing`
- 판단:
  - 현재 import 범위 안에 부족 / 씨족 / 유목 공동체 / 토착 마을 연합이 거의 보이지 않는다.
  - 오벨리스크는 생존 공동체와 군사 전선, 암시장, 망명 네트워크가 부족층 자리를 대체한다.

## 3. Guild / Economy Layer

- 상태: `enough`
- 판단:
  - 무역 문서와 세력 문서 모두에서 거래 구조가 매우 선명하다.
  - 특히 `본 마켓`, `잊힌 시장`, `서플라이 포트 알파`, 방벽 통행권, 기억 저당, 유물 거래, 불법 계약서가
    경제 층을 살아 있게 만든다.

## 4. Human Story Reading

오벨리스크의 핵심은 `신화적 파멸`보다 아래 층의 인간 비용이다.

- 기억을 지우느냐, 붙들고 사느냐
- 나라에서 버려진 최전선 병사가 어떻게 버티느냐
- 이름을 버린 망명자가 어떤 새 계약으로 살아가느냐
- 죽은 자가 기억을 화폐로 삼는 사회에서 무엇을 잃느냐

즉 이 대륙은 인간 서사 중심 기획과 충돌하지 않는다.
오히려 `붕괴 이후에도 관계, 책임, 생존 방식이 남아 있는 대륙`으로 쓰면 강하다.

## 5. Spatial Reading

공간 구조도 아주 강하다.

- 중앙 흑탑 지대
- 서부 사령평원
- 동부 황무지 유적
- 보이지 않는 결계선
- 오벨리스크 / 망자의 성채 / 질서의 탑 / 경계의 보루
- 네크로폴리스 프라임 / 본 마켓 / 잊힌 시장 / 서플라이 포트 알파

즉 오벨리스크는 나중에

- 대륙 지도
- 전선 지도
- 요새 지도
- 기록탑 / 제단 / 회랑 내부 맵

으로 확장하기 좋은 축이다.

## Conductor Decision

오벨리스크는 다음처럼 고정하는 것이 가장 합리적이다.

- `가문층`: thin
- `부족층`: missing
- `길드층`: enough
- `정체성`: 붕괴 후에도 기억, 거래, 방어선, 망명 네트워크가 남아 있는 대륙
- `활용 우선축`: 전선, 생존, 기억 거래, 버려진 병사, 지워진 이름, 봉인 유지 비용
