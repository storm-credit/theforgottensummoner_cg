# Setting Book Chapter 1 - Five Continents Draft

이 장은 `1~5 대륙`을 설정집 본문에서 읽기 위한 1차 초안이다.

핵심 판단은 단순하다.

5대륙은 서로 같은 방식으로 서지 않는다.
그 비대칭성이 이 세계의 장점이다.

## 1.0 Five-Continent Reading Rule

각 대륙은 `가문 / 부족 / 길드` 층을 모두 같은 밀도로 갖지 않는다.

설정집은 이 차이를 빈칸처럼 보정하지 않고,
각 대륙의 인간 서사 무대가 어떻게 다른지 보여주는 방식으로 쓴다.

요약하면 아래와 같다.

| Continent | Strong Layer | Thin / Support Layer | Human Story Direction |
|---|---|---|---|
| `에테르` | 가문, 길드, 도시 문명 | 대륙 공통 부족층 | 계승, 신분, 가문 책임, 도시 권력 |
| `크림슨` | 부족, 생존 연합, 사막 교역 | 전통 국가형 가문층 | 혈맹, 명예, 생존, 약탈과 거래 |
| `프로스트` | 부족, 클랜, 혹한 생존 경제 | 정주 귀족층 | 결속, 희생, 생존 비용 |
| `오벨리스크` | 비정통 엘리트, 기억 경제, 생존 조직 | 전통 가문 / 부족층 | 상실, 망명, 기억, 생존 계약 |
| `해양` | 가문, 함대, 항로, 길드 경제 | 토착 공동체층 | 상속, 항해 욕망, 해상 질서 |

결손층은 새 설정 발명으로 채우지 않는다.

이후 보강은 `audit/Five_Continent_Missing_Layer_Master_Lock.md`의
`policy -> evidence register -> first pass -> second pass -> overread firewall`
순서를 따른다.

## 1.1 에테르 대륙

에테르는 가문과 길드가 중심 문명을 이루고,
정령연합이 부족 질서를 보존하는 중심 문명 대륙이다.

대표 인간 군상:

- 왕가와 귀족가 사람들.
- 자유도시 상인.
- 마법협회 실무자.
- 길드 운영층.
- 숲 경계의 부족 공동체.

대표 갈등:

- 계승권 경쟁.
- 도시 권력과 자연권의 충돌.
- 왕국 질서와 길드 실익의 충돌.
- 문명 중심부와 변방 정체성의 긴장.

사회층 판정:

- 가문층: `enough`
- 부족층: `thin`
- 길드층: `enough`

에테르의 가문층은 성국과 왕국연합 축이 강하게 뒷받침한다.
길드층도 자유도시연합, 국제 길드 연합, 상단, 정보단, 영약회 축으로 충분히 선다.

다만 대륙 공통 부족층은 약하다.
정령연합 내부에서는 부족 영역, 부족장, 수인족 연합, 숲 구역 자치가 선명하지만,
그것을 에테르 전체의 `tribe_clan enough`로 올리지는 않는다.

설정집 safe read:

- 에테르는 중심 문명 대륙이다.
- 정령연합 내부 부족축은 강하지만, 에테르 전체 부족층은 `frontier echo / border support`까지만 허용한다.
- 인간 서사의 기본 정서는 질서 속 경쟁, 책임, 품위의 무게다.

## 1.2 크림슨 대륙

크림슨은 사막 생존과 씨족 정치,
약탈 경제와 교역 질서가 함께 서는 유목 대륙이다.

대표 인간 군상:

- 부족장 가문.
- 용혈 상층.
- 유목 상인.
- 전사 파견자.
- 사막 생존 공동체.

대표 갈등:

- 씨족 명예와 시장 타협의 충돌.
- 물길과 교역로를 둘러싼 생존 경쟁.
- 용혈 상층과 하층민의 격차.
- 혈맹과 배신.

사회층 판정:

- 가문층: `thin`
- 부족층: `enough`
- 길드층: `enough`

크림슨은 붉은 사막 부족 연합과 용의 후예를 통해
부족/씨족 구조가 강하게 확인된다.
사막 상인 연합, 유목 교역단, 전사 파견 조합, 카르텔, 상인회도 실제로 선다.

다만 전통 귀족국가형 가문층은 강하지 않다.
크림슨의 가문층은 왕가/귀족가보다
씨족장, 용혈 지배층, 부족 귀족에 가깝다.

설정집 safe read:

- 크림슨은 부족 strong 대륙이다.
- 국가형 가문층은 `씨족형 지배층과 분리된 thin-support`까지만 허용한다.
- `state_house strong`으로 과독하지 않는다.
- 정서는 갈증, 열기, 명예 경쟁, 생존 본능이다.

## 1.3 프로스트 대륙

프로스트는 혹한 생존과 클랜 정치,
보급 경제가 맞물린 북방 생존 대륙이다.

대표 인간 군상:

- 부족장 가문.
- 이동 공동체 전사.
- 사냥꾼.
- 방한 물자 상인.
- 생존 장비를 다루는 실무자.

대표 갈등:

- 열량과 보급을 둘러싼 경쟁.
- 클랜 충성심과 생존 거래의 충돌.
- 고립된 공동체와 외부 교역의 긴장.
- 부족장 책임과 폭력의 무게.

사회층 판정:

- 가문층: `thin`
- 부족층: `enough`
- 길드층: `enough`

프로스트본 연합은 부족 체계를 강하게 세운다.
동시에 클랜 상층, 상인 협회, 방어 협회, 얼음 독점단 같은
혹한 생존 경제 조직도 보인다.

그러나 프로스트의 가문층은 정주 귀족국가형보다
부족장 가문과 클랜 상층에 가깝다.

설정집 safe read:

- 프로스트는 부족/클랜 strong 대륙이다.
- 정주 귀족층은 요새, 보급, 병기소 운영층에서만 `thin-support`로 읽는다.
- 클랜 상층과 정주 귀족층을 곧바로 합치지 않는다.
- 정서는 혹한 압박, 인내, 연대, 잔혹한 생존 감각이다.

## 1.4 오벨리스크 대륙

오벨리스크는 문명이 붕괴한 뒤에도
기억, 봉인, 보급, 망명 네트워크가 살아남은 붕괴 후 대륙이다.

대표 인간 군상:

- 버려진 병사.
- 기억 거래상.
- 망명자.
- 봉인 수호자.
- 생존 거점 관리자.

대표 갈등:

- 기억을 지우고 버틸지 붙들고 무너질지의 선택.
- 봉인 유지 비용과 생존 비용의 충돌.
- 망명 네트워크 간 통치권 경쟁.
- 이름을 잃은 자의 계약.

사회층 판정:

- 가문층: `thin`
- 부족층: `missing`
- 길드층: `enough`

오벨리스크는 전통 귀족 사회보다
기억 귀족, 기록 상층, 망명 실권층, 계약 엘리트 같은
비정통 엘리트 구조가 강하다.

부족/씨족 기반은 거의 보이지 않지만,
기억 거래, 암시장, 보급항, 봉인 유지 비용, 유물 거래, 저당 경제 축은 강하다.

설정집 safe read:

- 오벨리스크는 붕괴 후 생존 조직과 비정통 엘리트의 대륙이다.
- `state_house` 신호는 먼저 `nontraditional elite thin-support`로 우회시킨다.
- monarchy / traditional state-house 복원은 금지한다.
- 정서는 폐허의 압박, 상실감, 망명, 생존 계약이다.

## 1.5 해양 대륙

해양은 항로, 함대, 해상 귀족, 해적 질서,
교단 권위가 뒤엉킨 해상 문명 대륙이다.

대표 인간 군상:

- 함대 귀족.
- 항만 상인.
- 선장.
- 해적 가문 구성원.
- 해양 성지를 지키는 신앙층.

대표 갈등:

- 항로 상속과 해상 귀족 경쟁.
- 함대와 해적의 질서 충돌.
- 교역 욕망과 교단 통제의 충돌.
- 자유와 신앙 사이의 줄다리기.

사회층 판정:

- 가문층: `enough`
- 부족층: `thin`
- 길드층: `enough`

해양은 항만, 함대, 교역, 교단 축이 강하고,
황금 함대 내부에서는 가문층도 확인된다.
해양 무역, 항만 상단, 암시장, 함대 경제, 교단 축도 강하게 선다.

반면 토착 공동체층은 아직 약하다.
항만/교단/함대 바깥 생활 관습의 support range로만 보존한다.

설정집 safe read:

- 해양은 항로와 함대, 해상 귀족과 시장이 강한 대륙이다.
- 토착 공동체층은 `local support outside fleet/church/port core`까지만 허용한다.
- `tribe_clan enough`로 승격하지 않는다.
- 정서는 개척 욕망, 불안정한 자유, 바다의 위압감, 항로 긴장이다.

## 1.6 Cross-Continent Balance

5대륙 균형은 아래처럼 읽는다.

- `가문 strong`: 에테르, 해양.
- `부족 strong`: 크림슨, 프로스트.
- `비정통 생존 엘리트 strong`: 오벨리스크.
- `길드 / economy strong`: 에테르, 해양, 오벨리스크.

이 구조는 각 대륙의 인간 군상을 다르게 만든다.

따라서 모든 대륙에 같은 종류의 가문,
같은 종류의 부족,
같은 종류의 길드를 억지로 넣지 않는다.

## 1.7 Missing Layer Lock

결손층 5개는 빈칸이 아니다.

이미 아래 순서로 읽도록 잠긴 정책 상태다.

1. Policy Lock.
2. Evidence Register.
3. First Pass A.
4. Second Pass A.
5. Overread Firewall.

설정집 본문에서는 아래 다섯 줄을 안전 기준으로 둔다.

| Continent | Safe Read | Overread Ban |
|---|---|---|
| `에테르` | outside `Spirit Union` 부족층은 `frontier echo / border support` | outside core를 부족 연합 본체로 복원 |
| `크림슨` | `tribe_clan core + state_house thin-support` | 세습 귀족국가 / `state_house strong` 복원 |
| `프로스트` | settled stronghold support layer only | settled nobility enough 승격 |
| `해양` | local support outside fleet/church/port core | tribe_clan enough 승격 |
| `오벨리스크` | state_house signal rerouted to nontraditional elite first | monarchy / traditional state-house 복원 |

## 1.8 Chapter Lock

이 장에서 고정하는 기준은 아래다.

- 5대륙은 서로 같은 방식으로 서지 않는다.
- 결손층은 실패가 아니라 정책적으로 얇게 허용된 층이다.
- 보강은 새 설정 발명이 아니라 admissible evidence intake로만 진행한다.
- 대륙 본체를 재정의할 권한은 summary 문서가 아니라 missing-layer master lock에 있다.
- 다음 장인 `2. 세력 아카이브 구조`는 이 대륙별 safe read를 넘지 않는다.
