# Five Continent Synthesis

이 문서는 현재까지 import한 작업본 기준으로
`1~5 대륙`의 인간 서사 무대 성격과 `가문 / 부족 / 길드` 결손을 한 장으로 묶은 종합 시트다.

결손층 본선 entry는
`audit/Five_Continent_Missing_Layer_Master_Lock.md`를 우선 기준으로 읽는다.

## Core Read

- `에테르`: 가문과 길드가 강한 중심 문명 대륙
- `크림슨`: 부족과 생존 연합이 강한 대륙
- `프로스트`: 부족과 혹한 연대가 강한 대륙
- `오벨리스크`: 붕괴 이후 생존 조직과 비정통 엘리트, 기억 경제가 강한 대륙
- `해양`: 항로, 항만, 함대, 귀족 연합, 해상 시장이 강한 대륙

즉 5대륙은 서로 같은 방식으로 서지 않는다.
이 비대칭성이 오히려 좋다.

## 1. Ether

- 강점:
  - 가문
  - 길드
  - 도시 / 왕국 / 협회 / 상단 구조
- 결손:
  - 부족 / 씨족 / 변방 공동체
- 인간 서사 방향:
  - 계승, 신분, 가문 책임, 도시 권력, 길드 경쟁

## 2. Crimson

- 강점:
  - 부족
  - 길드 / 상단 / 카르텔
  - 연합 / 족장 / 사막 생존 구조
- 결손:
  - 전통 가문
- 인간 서사 방향:
  - 생존, 혈맹, 명예, 유목, 약탈 경제, 씨족 계승 전통

## 3. Frost

- 강점:
  - 부족
  - 길드 / 독점 조직
  - 혹한 공동체, 연합, 전사 문화
- 결손:
  - 전통 가문
  - 도시 길드
- 인간 서사 방향:
  - 결속, 희생, 추위 속 생존, 클랜 정치, 혹한 생존 경제

## 4. Obelisk

- 강점:
  - 비정통 엘리트
  - 기억 거래 / 유물 거래 / 보급 경제
  - 전선 / 요새 / 봉인 / 생존 거점
- 결손:
  - 전통 가문
  - 부족 / 씨족 / 토착 공동체
- 인간 서사 방향:
  - 버려진 병사, 망명자, 죽은 자의 기억, 이름을 잃은 자의 계약

## 5. Oceanic

- 강점:
  - 가문
  - 길드
  - 항구 / 함대 / 해적 / 교단 / 교역
- 결손:
  - 토착 공동체 / 씨족 / 해양 원주민층
- 인간 서사 방향:
  - 항로, 상속, 해상 귀족 경쟁, 해적 질서, 항구 도시 욕망

## Cross-Continent Balance

현재 5대륙 균형은 다음처럼 읽는 게 가장 자연스럽다.

- `가문 strong`: 에테르, 해양
- `부족 strong`: 크림슨, 프로스트
- `비정통 생존 엘리트 strong`: 오벨리스크
- `길드/economy strong`: 에테르, 해양, 오벨리스크

이 구조는 각 대륙의 인간 군상을 다르게 만들기 때문에 장점이 있다.
굳이 모든 대륙을 비슷한 밀도로 맞출 필요는 없다.

다만 아래 결손은 `빈칸`이 아니라 `정책적으로 얇게 허용된 층`으로 남는다.

- 에테르의 부족층은 `정령연합 outside core`에서 `frontier echo / border support`
- 크림슨의 전통 가문층은 `씨족형 지배층과 분리된 thin-support`
- 프로스트의 전통 가문층은 `정주 거점 운영층 thin-support`
- 해양의 토착 공동체는 `support range`
- 오벨리스크의 가문층은 `nontraditional elite thin-support`

이 얇은 층 5개가 실제로 어떤 신호를 받을 수 있는지는
`audit/Five_Continent_Missing_Layer_Evidence_Register.md`를
증거 intake 기준으로 함께 본다.

## Conductor Decision

다음부터는 `새로운 강점 발굴`보다
`이미 잠근 얇은 층을 과독하지 않게 유지하면서 증거를 더 모으는 것`을 메인 목표로 둔다.

우선순위:

1. 에테르 outside `정령연합` 부족층 support 증거 확인
2. 크림슨 국가형 가문층 thin-support 증거 확인
3. 프로스트 정주 귀족층 thin-support 증거 확인
4. 해양 토착 공동체 support-range 증거 확인
5. 오벨리스크 nontraditional elite 정책 반영 여부 재검산

이 문서를 이후 `8번 세력`, `14번 영웅`, `15번 중간 등장인물`, 지도 축의 기준판으로 사용한다.

단, 결손층 보강은 이 종합 시트에서 바로 승격하지 않고,
먼저 `policy lock + evidence register` 범위 안에서만 받아 적는다.
