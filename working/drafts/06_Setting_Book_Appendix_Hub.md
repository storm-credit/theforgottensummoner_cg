# Setting Book Appendix Hub

## 부록 기준으로 보고 싶으면 여기서 시작

이 문서는 설정집 부록을 기준으로 보기 위한 허브다.
본문을 읽다가
"이건 왜 아직 확정이 아니지?"
"이 이름은 왜 함부로 합치면 안 되지?"
"이 유물은 왜 아직 후보 상태지?"
같은 질문이 생길 때 여기로 들어오면 된다.
