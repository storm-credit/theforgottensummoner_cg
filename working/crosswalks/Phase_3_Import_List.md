## Why This Batch

- 현재 import 기준으로는 `에테르 대륙`만 실감사가 가능하다.
- `에테르`는 가문층과 길드층이 강하게 보이므로, 실제 `8번` 구조와 연결해야 한다.
- `크림슨 / 프로스트`는 부족층 검증 우선도가 높다.
