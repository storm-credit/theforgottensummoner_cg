# Faction Naming Replacement Register

이 문서는 현대적이거나 범죄물/기업물처럼 들리는 조직명을
정통 판타지 톤의 `display canon` 후보명으로 바꾸기 위한 등록부다.

중요한 점:

- 이 문서는 파일명 일괄 수정안이 아니다.
- 원본 경로는 건드리지 않는다.
- 현재는 `표면 정본명 후보`만 모은다.
- 내부 기능 분류와 표면 명칭은 분리한다.

## Replacement Rule

1. 한국어 명칭이 이미 충분히 판타지 톤이면 그대로 둔다.
2. 현대적인 것은 우선 영문 suffix부터 교체를 검토한다.
3. 한국어 자체가 현대적이면 한국어 표면명도 새 후보를 만든다.
4. 아직 근거가 약하면 `Soft Canon`으로 둔다.

## Conductor Note

현재 가장 먼저 교체 검토할 이름은 아래 둘이다.

1. `철의 금융 카르텔`
2. `침묵의 카르텔`

이 둘은 한국어 표면명까지 현대 어감이 강해서,
나중에 `14번`, `15번`, 관계망에 연쇄적으로 영향을 줄 가능성이 크다.

## Next Use

- `14번` 고아 인물의 소속 표기를 쓸 때 이 등록부를 우선 참고한다.

- 실제 표면 정본명 확정은 대륙별 어계 규칙이 더 잡힌 뒤에 한다.
