# Supranational Reference Bridge

이 문서는 `14번` 인물 문서와 고아 인물 메모에서 자주 인용되는
`범대륙 초국가 및 중립` 참조명을
현재 `display canon` 후보와 연결하는 다리 문서다.

원칙은 간단하다.

- 원본 위키링크와 파일 경로는 그대로 둔다.
- 하지만 요약, 관계표, 소속 서술, 인물 카드에서는 아래 display canon을 우선 쓴다.
- 즉 `참조 경로`와 `표면 표시명`을 분리 관리한다.

## Direct Application Notes

### Hero Summary Writing

영웅 요약 카드에서 다음처럼 쓴다.

- `국제 길드 연합의 규정망 아래 관리됨`
- `대륙 무역 연맹과 결탁한 유통권 인물`
- `마도 공학 대공방 연합의 양산 체계와 연결됨`
- `그림자 첩보망이 추적하거나 보호하는 대상`

### Orphan Character Writing

`[현존] 카르텔 수장 칼리크 디트리히` 같은 경우는
아래 식으로 쓴다.

- 기능 분류:
  - `금융 연맹형 조직`
- display candidate:
  - `철의 금융 연맹`
  - `침묵의 상회`
- reference anchor:
  - `6-5. 철의 금융 카르텔 (Iron Bank)`
  - 관련 하위 카르텔 개요 파일

### Contact Table Writing

접점 표에서는 경로 대신 display canon을 우선 쓴다.

예:

- `국제 길드 연합 규정망`
- `대륙 무역 연맹 후원권`
- `그림자 첩보망 관측 대상`

## Conductor Note

이 문서의 목적은 이름을 즉시 바꾸는 것이 아니라,
다음 단계 문서들이 현대적인 경로명을 그대로 드러내지 않도록 완충재를 두는 것이다.
