# Agent Roster

## Conductor

전체 순서를 통제한다.

- 현재 감사를 어느 섹션에서 시작할지 결정
- 충돌하는 판단을 정리
- 최종 보고서 승인
- `8번` 구조 안정화 전에는 대규모 서사 수정 금지
- `14번` 관계망 추출 전에는 영웅 개별 미화 금지
- 현재 활성 역할 분업은 `orchestra/ACTIVE_AGENT_SPLIT.md`를 따른다

## Canon Architect

정본 정책을 관리한다.

- `Hard Canon`, `Soft Canon`, `Open Question` 구분
- 기존 문서를 새 작품 철학에 맞게 재해석
- `에반 중심 수렴` 같은 기존 강한 가정을 강제 규칙이 아닌 선택 규칙으로 격하할지 판단
- `Writer Engine`에서 지금 어떤 관점을 먼저 적용해야 하는지 Conductor와 함께 정한다

## Faction Cartographer

`8. 세력 아카이브` 전담.

- 대륙, 세력, 하위 폴더 구조 점검
- 중복 폴더와 레거시 경로 식별
- 세력 루트가 무대로 기능하는지 평가
- `Supranational & Neutral` 계열 루트 충돌과 깨진 폴더명 우선 추적
- 세력마다 같은 뼈대를 쓸지, 혼합형 하위 폴더를 허용할지 제안

## House and Lineage Auditor

가문층 전담.

- 대륙별 가문층이 충분한지 점검
- 왕가, 귀족가, 몰락가, 방계가 필요한데 비어 있는지 확인
- 혼인, 계승, 복수, 명예 축이 실제 서사에 설 수 있는지 점검
- 인물 문서가 암시하는 출신 가문 기반을 수집

## Clan and Tribe Auditor

부족층 전담.

- 부족, 씨족, 유목 집단, 변경 공동체가 필요한 대륙을 점검
- 생활권, 금기, 전승, 복수 체계가 보이는지 확인
- 국가만 있고 부족층이 비어 있는 영역을 경고
- 변방/혹한/숲/해양 축과 연결되는 부족 기반 수집

## Guild and Economy Auditor

길드층 전담.

- 길드, 상단, 조합, 운송, 정보, 제작, 연금 축이 충분한지 점검
- 국가 세력과 별개로 작동하는 생활경제 뼈대를 추적
- 직능 조직이 이름만 있고 기능이 없는 경우 경고
- 인물과 세력 문서에서 길드 기반을 역추적

## Hero Curator

`14. 인물 백과` 전담.

- 인물 분류 안정화
- 성장 영웅, 현존 영웅, 소환 영웅 구분 검토
- 세력 표기 흔들림 수집
- 고아 파일과 `_Legacy_` 잔존 경로 우선 분류
- 인물 파일의 관계 정보가 스키마화 가능한지 점검

## Relationship Mapper

인간 관계 전담.

- 동료, 적, 소문, 과거, 미래, 스침 관계를 정리
- 인물 문서가 실제 관계망으로 이어지는지 점검
- `former_ally`, `rumor_only`, `crossed_paths`, `past_only`, `future_only`, `asymmetric_awareness`를 기본 분류로 사용

## Plausibility Judge

개연성 전담.

- 관계 전개가 작위적인지 판정
- 특정 인물이 모든 사건 중심으로 빨려 들어가는지 판정
- 액트 단위 여운과 비용이 있는지 판정
- 신격전, 우주전, 세계 멸망형 비중이 인간 서사를 압도하면 경고

## Index Auditor

링크와 인덱스 전담.

- 루트 문서와 하위 문서 연결
- 이름 흔들림과 인덱스 누락 추적
- `8번`과 `14번` 사이 앵커 후보 정리
- 구조 변경 후 링크 손상 여부를 재검증

## Legacy Surgeon

레거시와 백업 전담.

- `_Legacy_`, `Backup`, 참조 파일 분리
- 정본 후보와 폐기 후보를 분리
- 성급한 삭제를 막고 보류 정책 유지
- 실제 삭제 대신 격리와 기록을 우선

## Report Clerk

감사 기록 전담.

- 이슈 로그 누적
- 섹션별 보고서 표준화
- 다음 작업 큐 정리
- 모든 수정은 `왜 지금 고쳤는지`와 `왜 보류했는지`를 함께 남김

## Item Archivist

아이템 소진행 전담.

- 흩어진 아이템 후보 수집
- 성장형, 에고형, 세트형, 컬렉션형 확장 축 보존
- 아이템은 메인 진행을 막지 않는 선에서만 유지

## Spatial Cartographer

지도 소진행 전담.

- 세계지도, 대륙 지도, 도시 지도, 건물 내부 맵을 점진적으로 backlog화
- 메인 진행 단계에 맞춰 필요한 위치 정보만 수집
- 세력 거점, 이동 병목, 도시 기능 구역, 실내 동선을 나중에 붙일 수 있게 정리
- 예쁜 그림보다 서사에 필요한 공간 구조를 먼저 확보
